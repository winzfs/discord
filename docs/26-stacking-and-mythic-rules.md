# 26. 중첩 이동 / 신화 중첩 제한 / 합성 배치 규칙

## 1. 목적

유닛 중첩과 합성 결과 배치를 더 직관적으로 만들기 위해 다음 규칙을 적용했습니다.

```text
일반~전설 유닛은 같은 유닛끼리 최대 3개까지 중첩
신화 유닛은 중첩 불가
중첩된 유닛을 이동하면 스택 전체가 함께 이동
영웅 합성 결과가 이미 필드에 있으면 가장 먼저 필드에 있던 동일 유닛 스택에 먼저 중첩
```

## 3. 보드 중첩 규칙

### 3.1 일반~전설

- 같은 heroId면 한 칸에 최대 3개까지 중첩 가능
- 이동 시 마지막 1개만 움직이지 않고 스택 전체가 이동
- 다른 유닛 칸과 드래그하면 스택끼리 자리 교환

### 3.2 신화

- 신화 등급은 기존 유닛 위에 중첩되지 않음
- 기존 신화가 있는 칸에도 다른 유닛을 중첩하지 않음
- 신화 유닛끼리도 중첩하지 않음

## 4. 합성 결과 배치 규칙

합성으로 새 유닛이 생성되면 다음 순서로 배치합니다.

```text
1. 합성 결과와 같은 heroId의 기존 필드 유닛 스택 탐색
2. 가장 먼저 발견된 스택이 있고 3개 미만이면 그곳에 중첩
3. 없으면 기존 placeHeroOnBoard 규칙에 따라 배치
4. 결과가 신화 등급이면 중첩하지 않고 빈 칸에 배치
```

## 5. 신화 조합 메뉴 표시 개선

신화 조합 메뉴에서 재료 줄이 잘리는 문제가 있어 row 높이를 늘렸습니다.

```text
ROW_HEIGHT 104 → 122
ROW_GAP 8 → 9
```

재료 라벨 줄 간격도 넓혔습니다.

```text
INGREDIENT_START_Y = 66
INGREDIENT_ROW_GAP = 25
```

## 6. 테스트 체크리스트

- 일반~전설 같은 유닛 2~3개가 한 칸에 중첩되는지 확인
- 신화 유닛은 어떤 방식으로도 중첩되지 않는지 확인
- 중첩된 유닛을 드래그하면 스택 전체가 이동하는지 확인
- 중첩된 유닛 스택끼리 다른 칸과 자리 교환되는지 확인
- 합성 결과가 이미 필드에 있는 유닛이면 기존 스택에 먼저 중첩되는지 확인
- 합성 결과가 신화면 기존 스택에 중첩되지 않는지 확인
- 신화 조합 메뉴 재료 글자가 덜 잘리는지 확인
