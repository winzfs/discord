# 14. 히어로 스트라이크 슈팅 모드

## 목적

기존 랜덤 디펜스와 퍼즐 모드에서 분리된 모바일 세로형 아케이드 슈팅게임입니다.

- 경로: `/hero-strike`
- 렌더링: React에서 호스팅하는 Canvas 2D
- 조작: 화면 상대 드래그 이동, 기본 공격 자동 발사
- 구성: 6개 독립 스테이지와 스테이지별 보스전
- 저장: 브라우저 `localStorage` 최고 점수

## 핵심 플레이

1. 화면을 끌어 트레이서를 이동합니다.
2. 기본 공격은 자동 발사됩니다.
3. 적이 떨어뜨린 경험치를 모으면 레벨이 오릅니다.
4. 레벨업 때 세 가지 강화 중 하나를 선택합니다.
5. 적 탄환을 가까이 스치면 `GRAZE` 보너스와 궁극기 게이지를 얻습니다.
6. 각 스테이지 제한 시간이 지나면 보스가 등장합니다.
7. 보스를 격파하면 `STAGE CLEAR` 화면이 열립니다.
8. 다음 스테이지 버튼을 누르면 회복과 보조 아이템을 받고 다음 지역으로 이동합니다.
9. 6스테이지 보스를 격파하면 전체 승리합니다.

## 스테이지 구성

일반전 시간은 스테이지마다 45초에서 70초이며, 총 일반전 시간은 약 5분 45초입니다. 보스전 시간은 별도입니다.

### Stage 1 — KING'S ROW

- 45초
- 러너와 기본 드론 중심
- 단일 조준탄
- 보스 패턴: 7방향 조준 부채꼴
- 보스: `SENTRY PRIME`

### Stage 2 — HANAMURA RUINS

- 50초
- 저격기와 지그재그 기체 등장
- 좌우 분열탄
- 보스 패턴: 회전하는 꽃잎형 원형탄
- 보스: `KITSUNE WATCHER`

### Stage 3 — NULL SECTOR FACTORY

- 55초
- 드론·탱커·폭격기 혼합
- 조준탄과 십자탄 동시 발사
- 보스 패턴: 양방향 나선탄
- 보스: `FACTORY WARDEN`

### Stage 4 — ANTARCTIC BASE

- 60초
- 저격기 비중 증가
- 세로로 내려오는 빗줄기형 탄막
- 보스 패턴: 빈 공간이 이동하는 레인 탄막
- 보스: `CRYO COMMANDER`

### Stage 5 — JUNKERTOWN SKYWAY

- 65초
- 폭격기와 지그재그 기체 비중 증가
- 넓은 부채꼴 탄막
- 보스 패턴: 고속 집중 연사
- 보스: `SCRAP COLOSSUS`

### Stage 6 — GIBRALTAR ORBIT

- 70초
- 모든 적이 고르게 등장하는 최종전
- 다중 조준탄과 고속 탄막
- 보스 패턴: 집중탄·나선탄·레인탄 순환
- 보스: `ORBIT OVERSEER`

스테이지 보스 처치 후 다음 스테이지로 바로 넘어가지 않습니다. 별도 클리어 화면에서 점수와 보상을 확인한 뒤 사용자가 화면을 눌러 진행합니다.

다음 스테이지 기본 보상:

- 체력 1 회복
- 보호막 1 획득
- 궁극기 게이지 25% 획득
- 스테이지별 보조 아이템

## 적 종류

- 러너: 빠르게 내려오며 직접 압박
- 드론: 기본 조준 사격
- 탱커: 높은 체력과 넓은 탄막
- 저격기: 상단에서 정지 후 빠른 바늘탄 발사
- 지그재그 기체: 좌우로 크게 움직이며 교차탄 발사
- 폭격기: 느리고 단단하며 넓은 중탄 투하
- 보스: 스테이지마다 전용 체력과 탄막 패턴 사용

## 아이템 종류

- 경험치 조각: 레벨업 경험치
- 회복 팩: 체력 1 회복
- 궁극기 코어: 궁극기 게이지 충전
- 방벽 코어: 보호막 1 획득
- 폭발 코어: 적탄 제거와 화면 전체 피해
- 오버드라이브 코어: 일정 시간 화력 강화
- 유도 미사일 코어: 15초간 가장 가까운 적을 추적하는 폭발 미사일 자동 발사
- 지원 드론 코어: 18초간 트레이서 양옆에서 추가 자동 사격
- 시간 왜곡 코어: 9초간 적 이동과 적탄 속도 감속
- 경험치 코어: 경험치 45 즉시 획득

유도 미사일은 적을 자동 재탐색하며, 착탄 시 주변 적에게 범위 피해를 줍니다. 보스에게는 범위 피해량이 일부 감소합니다.

## 피격 판정

- 트레이서 스프라이트 표시 크기: 기존과 동일
- 실제 피격 반경: 18에서 9로 축소
- 화면 중앙의 작은 청록색 원이 실제 피격점
- 아이템 수집 반경은 기존 크기를 유지해 수집 조작감이 나빠지지 않도록 분리

## 시인성 원칙

적 탄환과 보상 아이템은 색상에만 의존하지 않습니다.

- 적 기본탄: 분홍·빨강 원형, 검은 외곽선, 흰색 중심점
- 저격탄: 길고 뾰족한 바늘 모양
- 교차탄: 마름모 파편 모양
- 중탄: 큰 원형과 밝은 외곽선
- 경험치: 청록색 다이아 형태와 아래쪽 빛 꼬리
- 회복: 녹색 십자
- 궁극기: 금색 번개
- 보호막: 파란 방패
- 폭탄: 주황 폭탄
- 오버드라이브: 보라색 별
- 유도 미사일: 주황색 로켓
- 지원 드론: 연두색 소형 기체
- 시간 감속: 청록색 시계
- 경험치 코어: 큰 청록색 다이아

색각 차이가 있거나 화면 밝기가 낮아도 형태만으로 종류를 구분할 수 있도록 구성합니다.

## 구현 기능

- 트레이서 기존 스프라이트 시트 재사용
- 자동사격과 다중 탄환
- 일반 적 6종과 스테이지 보스
- 6개 스테이지별 적 편성, 배경, 탄막 속도와 패턴
- 6종의 보스 전용 탄막 패턴
- 스테이지 클리어와 다음 스테이지 화면
- 체력 4칸과 시작 보호막
- 축소된 플레이어 피격점과 기존 아이템 수집 범위
- 경험치 드롭과 자석 흡수
- 특수 아이템 9종
- 유도 미사일과 지원 드론 런타임
- 시간 감속 효과
- 3택 레벨업 강화 8종
- 콤보 점수 배율
- 그레이즈 판정
- 오버드라이브
- 펄스 폭탄 궁극기
- 보스 경고와 보스 HP 바
- 시작, 일시정지, 스테이지 클리어, 게임오버, 승리, 재시작 화면
- 파티클, 화면 흔들림, 플래시, 부유 점수 연출
- 최대 1.5배 `devicePixelRatio` 렌더링
- 모바일 세이프 에어리어와 화면 비율 대응

## 모바일 성능 기준

- 스테이지별 정적 배경은 캔버스에 미리 그린 뒤 재사용합니다.
- 탄환과 아이템은 고비용 `shadowBlur` 대신 단순한 도형으로 표시합니다.
- 탄환은 최대 220개, 유도 미사일은 최대 16개, 파티클은 최대 140개로 제한합니다.
- 별 배경 개수는 42개로 제한합니다.
- 드래그는 손가락이 움직인 거리만큼 상대 이동합니다.
- 브라우저가 묶어서 전달한 포인터 이벤트에서는 가장 최신 좌표를 사용합니다.

## 리소스 원칙

새 캐릭터 이미지를 추가하지 않고 기존 트레이서 스프라이트를 사용합니다.

```text
/assets/heroes/tracer.png
```

일반 적과 보스, 아이템은 기존 게임과 같은 도형 기반 아트 방향으로 Canvas 2D에서 표현합니다.

## 배포 후 점검 항목

- 6개 스테이지의 제한 시간과 보스 등장
- 각 보스 처치 후 클리어 화면과 다음 스테이지 이동
- 신규 적 3종의 이동과 탄막 패턴
- 스테이지별 일반탄과 보스탄의 차이
- 유도 미사일 재탐색, 범위 피해와 보스 피해
- 지원 드론 추가 사격과 지속 시간
- 시간 감속 중 적·적탄만 느려지는지
- 신규 아이템 아이콘과 수집 효과
- 작은 피격점과 그레이즈 판정
- 6스테이지 후반까지 프레임 유지
- 레벨업 직후 스테이지 흐름이 멈추거나 중복되지 않는지
