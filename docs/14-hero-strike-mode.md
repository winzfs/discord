# 14. 히어로 스트라이크 슈팅 모드

## 목적

기존 랜덤 디펜스와 퍼즐 모드에서 분리된 모바일 세로형 아케이드 슈팅게임입니다.

- 경로: `/hero-strike`
- 렌더링: React에서 호스팅하는 Canvas 2D
- 조작: 화면 상대 드래그 이동, 기본 공격 자동 발사
- 구성: 6개 독립 스테이지와 스테이지별 보스전
- 성장: 경험치 레벨업과 3택 영구 강화
- 저장: 브라우저 `localStorage` 최고 점수

## 핵심 플레이

1. 화면을 끌어 트레이서를 이동합니다.
2. 기본 공격은 자동 발사됩니다.
3. 경험치를 모으면 레벨이 오릅니다.
4. 레벨업 때 세 가지 강화 중 하나를 선택합니다.
5. 선택한 무기와 능력은 해당 플레이 동안 영구 유지됩니다.
6. 각 스테이지 제한 시간이 지나면 보스가 등장합니다.
7. 보스를 격파하면 클리어 화면에서 보상을 받고 다음 지역으로 이동합니다.
8. 6스테이지 보스를 격파하면 전체 승리합니다.

## 스테이지 구성

일반전은 45초부터 70초까지이며 보스전 시간은 별도입니다.

| 스테이지 | 일반전 | 일반 탄막 | 보스 패턴 |
|---|---:|---|---|
| KING'S ROW | 45초 | 단일 조준탄 | 7방향 부채꼴 |
| HANAMURA RUINS | 50초 | 좌우 분열탄 | 회전 꽃잎탄 |
| NULL SECTOR FACTORY | 55초 | 조준·십자탄 | 양방향 나선탄 |
| ANTARCTIC BASE | 60초 | 빗줄기 탄막 | 이동 레인 탄막 |
| JUNKERTOWN SKYWAY | 65초 | 넓은 부채꼴 | 고속 집중 연사 |
| GIBRALTAR ORBIT | 70초 | 고속 다중탄 | 집중·나선·레인 혼합 |

다음 스테이지 진입 시 체력 1, 보호막 1, 궁극기 25%와 스테이지 보상 아이템을 받습니다.

## 적 종류

- 러너: 빠르게 내려오며 직접 압박
- 드론: 기본 조준 사격
- 탱커: 높은 체력과 넓은 탄막
- 저격기: 상단에서 빠른 바늘탄 발사
- 지그재그 기체: 좌우로 움직이며 교차탄 발사
- 폭격기: 느리고 단단하며 대형 중탄 투하
- 보스: 스테이지별 전용 체력과 탄막 패턴 사용

## 레벨업 강화

레벨업마다 세 가지 선택지가 표시됩니다. 첫 레벨업에는 유도 미사일이 반드시 포함되며, 이후에도 최소 한 개의 무기 계열 선택지가 제공됩니다.

### 기본 화력

- 속사 모듈: 공격 간격 18% 감소, 최대 5단계
- 펄스 확장: 정면 탄환 수 증가, 최대 3단계
- 고출력 코어: 기본탄·미사일·드론의 기준 공격력 증가
- 관통 탄환: 기본탄 관통 횟수 증가
- 오버클럭: 오버드라이브 추가 피해 증가
- 치명 코어: 치명타 확률과 치명타 배율 증가

### 영구 보조 무기

- 유도 미사일: 가장 가까운 적을 자동 추적하고 착탄 시 범위 피해
  - 1단계에서 영구 해금
  - 재선택 시 발사 간격, 비행 속도, 추적 성능, 피해와 폭발 범위 증가
  - 최대 5단계
- 드론 편대: 트레이서 양옆에서 영구 자동 사격
  - 단계가 오르면 공격력과 연사 속도 증가
  - 4단계에서 관통 1회 추가
- 측면 포대: 좌우 대각선 방향으로 보조탄 발사
  - 최대 3단계
- 후방 방어포: 아래쪽으로 방어탄 발사
  - 2단계부터 세 갈래 후방 사격
- 폭발 탄두: 기본탄과 보조탄이 명중한 주변에 추가 피해
  - 최대 4단계
- 연쇄 코어: 명중 피해가 가까운 적에게 순차 전이
  - 최대 3단계

폭발탄·연쇄 코어·치명 코어는 기본탄뿐 아니라 지원 드론과 일부 보조 무기에도 함께 적용돼 조합형 빌드를 만들 수 있습니다.

### 생존과 편의

- 회수 자석: 경험치 흡수 범위 증가
- 시간 방벽: 보호막 1칸 획득
- 펄스 드라이브: 궁극기 게이지 즉시 충전

## 필드 아이템

유도 미사일은 필드 아이템으로 등장하지 않습니다. 레벨업 강화로만 획득합니다.

- 경험치 조각: 레벨업 경험치
- 회복 팩: 체력 1 회복
- 궁극기 코어: 궁극기 게이지 충전
- 방벽 코어: 보호막 1 획득
- 폭발 코어: 적탄 제거와 화면 전체 피해
- 오버드라이브 코어: 일정 시간 화력 강화
- 지원 드론 코어: 18초간 임시 드론 편대 활성화 또는 영구 드론 강화
- 시간 왜곡 코어: 9초간 적 이동과 적탄 속도 감소
- 경험치 코어: 경험치 45 즉시 획득

## 피격 판정

- 트레이서 스프라이트 표시 크기: 72px 유지
- 실제 피격 반경: 9
- 캐릭터 중앙의 작은 청록색 원이 실제 피격점
- 아이템 수집 반경은 외형 기준으로 유지

## 시인성 원칙

- 적 기본탄: 분홍·빨강 원형과 흰 중심점
- 저격탄: 길고 뾰족한 바늘
- 교차탄: 마름모 파편
- 중탄: 큰 원형과 밝은 외곽선
- 경험치: 청록 다이아와 빛 꼬리
- 유도 미사일: 주황색 로켓과 추진 불꽃
- 드론 편대: 연두색 소형 기체
- 시간 감속: 청록색 원형 오라

색상만이 아니라 형태로도 위험물과 보상을 구분합니다.

## 구현 기능

- 6개 스테이지와 6종 보스 패턴
- 일반 적 6종
- 영구 무기 포함 레벨업 강화 15종
- 첫 레벨업 유도 미사일 확정 선택지
- 유도 미사일 자동 재탐색과 범위 폭발
- 영구·임시 지원 드론
- 측면 포대와 후방 방어포
- 폭발탄, 연쇄 피해, 치명타
- 콤보, 그레이즈, 오버드라이브, 궁극기
- 작은 피격점과 별도 아이템 수집 범위
- 최고 점수 저장

## 모바일 성능 기준

- 탄환 최대 220개
- 유도 미사일 최대 16개
- 파티클 최대 140개
- 부유 텍스트 최대 10개
- 최대 1.5배 `devicePixelRatio` 렌더링
- 정적 배경 캐시
- 고비용 `shadowBlur` 미사용

## 배포 후 점검 항목

- 첫 레벨업에 유도 미사일 카드가 표시되는지
- 유도 미사일이 선택 후 시간 제한 없이 계속 발사되는지
- 미사일 재선택 시 연사와 폭발 범위가 증가하는지
- 측면·후방 포대의 발사 방향
- 폭발 피해와 연쇄 피해가 같은 적에게 중복 적용되지 않는지
- 치명타가 기본탄·드론·미사일에 적용되는지
- 스테이지 전환 후 영구 무기가 유지되는지
- 후반 탄막에서도 프레임이 유지되는지
