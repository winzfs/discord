# 14. 히어로 스트라이크 슈팅 모드

## 목적

기존 랜덤 디펜스와 퍼즐 모드에서 분리된 모바일 세로형 아케이드 슈팅게임입니다.

- 경로: `/hero-strike`
- 렌더링: React에서 호스팅하는 Canvas 2D
- 조작: 화면 상대 드래그 이동, 기본 공격 자동 발사
- 구성: 10개 독립 스테이지와 스테이지별 보스전
- 성장: 경험치 레벨업, 스테이지 프로토콜, 전투 등급
- 저장: 브라우저 `localStorage` 최고 점수

## 핵심 플레이

1. 화면을 끌어 트레이서를 이동합니다.
2. 기본 공격과 보조 무기는 자동 발사됩니다.
3. 적이 떨어뜨린 경험치를 모아 레벨업 강화 3택을 선택합니다.
4. 보스를 쓰러뜨리면 작전 프로토콜 3택 중 하나를 선택합니다.
5. 프로토콜을 선택하면 누적 성장한 상태로 다음 스테이지가 시작됩니다.
6. 3·6·9스테이지를 클리어하면 전투 등급이 상승합니다.
7. 10스테이지 최종 보스를 격파하면 전체 승리합니다.

- 적 탄환 속도
- 탱커와 폭격기 비율
- 보스 기본 체력과 레벨당 추가 체력
- 클리어 점수와 보스 점수

## 레벨업 강화

전투 중 경험치를 모으면 기존 레벨업 강화 3택이 표시됩니다.

- 속사 모듈
- 펄스 확장
- 고출력 코어
- 관통 탄환
- 회수 자석
- 시간 방벽
- 펄스 드라이브
- 오버클럭
- 유도 미사일
- 드론 편대
- 측면 포대
- 후방 방어포
- 폭발 탄두
- 연쇄 코어
- 치명 코어

모든 강화는 현재 단계, 다음 단계, 최대 단계를 표시하고 재선택할수록 실제 수치가 증가합니다.

## 작전 프로토콜

보스 처치 후 다음 스테이지로 바로 넘어가지 않습니다. 클리어 화면에서 세 가지 작전 프로토콜 중 하나를 선택해야 합니다.

각 프로토콜은 최대 3단계까지 누적됩니다.

### 생체 코어

- 최대 체력 +1
- 단계가 오를수록 선택 즉시 회복량 증가

### 반응로 증폭

- 기본탄, 측면·후방 포대, 드론, 유도 미사일, 폭탄과 궁극기의 피해 증가
- 선택할 때마다 누적 12% 증가

### 전투 데이터

- 모든 경험치 조각과 경험치 코어 획득량 증가
- 단계당 18% 증가

### 정밀 링크

- 기존 치명 코어와 별도로 추가 치명타 확률과 배율 제공
- 단계당 치명타 확률 5%, 치명타 배율 0.1 증가

### 펄스 동기화

- 기본 공격, 드론, 유도 미사일의 자동사격 간격 감소
- 선택할 때마다 누적 8% 감소

### 회수 배열

- 기존 회수 자석과 별도로 경험치 흡수 범위 추가
- 단계당 자석 범위 +70
- 선택 즉시 단계에 따른 보호막 획득

### 현상금 네트워크

- 적 처치와 보스 처치 점수 증가
- 단계당 20% 증가
- 콤보 점수 배율과 곱연산

## 전투 등급

3·6·9스테이지 클리어 후 다음 스테이지에 진입할 때 전투 등급이 상승합니다.

등급 상승 보너스:

- 전체 무기 피해 8% 증가
- 모든 자동사격 간격 5% 감소
- 최대 체력 +1
- 체력 완전 회복
- 보호막 +2
- 궁극기 완전 충전
- 등급에 따른 추가 점수

전투 등급은 HUD에 `RANK`로 표시됩니다.

## 성장 계층 분리

성장 요소는 서로 덮어쓰지 않도록 세 계층으로 계산합니다.

1. 기본 플레이어 수치
2. 경험치 레벨업 강화
3. 작전 프로토콜과 전투 등급 배율

예를 들어 고출력 코어를 나중에 선택해도 반응로 증폭과 전투 등급의 공격력 보너스가 사라지지 않습니다. 치명 코어와 정밀 링크, 회수 자석과 회수 배열도 합산됩니다.

## 스테이지 클리어 흐름

1. 보스 처치
2. 적 탄환과 남은 적 정리
3. 클리어 점수 지급
4. 작전 프로토콜 3택 표시
5. 프로토콜 선택 및 적용
6. 3·6·9스테이지면 전투 등급 상승
7. 체력 1, 보호막 1, 궁극기 보상 지급
8. 다음 스테이지 보조 아이템 지급
9. 다음 작전 시작

최종 10스테이지에서는 프로토콜 선택 없이 바로 전체 승리 화면으로 이동합니다.

## 경험치 조각 이동

- 생성 순간부터 아래 방향 속도를 가집니다.
- 생성 후 0.65초 동안 자석을 무시하고 강제로 낙하합니다.
- 강제 낙하 이후에만 자석 범위에서 플레이어를 추적합니다.
- 자석 범위 밖에서는 아래 방향으로 계속 가속됩니다.
- 경험치 조각 수명은 12초입니다.

## 피격 판정

- 트레이서 표시 크기: 72px 유지
- 실제 피격 반경: 9
- 캐릭터 중앙의 작은 청록색 원이 실제 피격점
- 아이템 수집 반경은 피격점과 별도로 계산

## 모바일 성능 기준

- 탄환 최대 220개
- 유도 미사일 최대 16개
- 파티클 최대 140개
- 부유 텍스트 최대 10개
- 최대 1.5배 `devicePixelRatio` 렌더링
- 정적 배경 캐시
- 고비용 `shadowBlur` 미사용

## 배포 후 점검 항목

- 시작 화면과 HUD가 10스테이지로 표시되는지
- 추가 4개 스테이지가 순서대로 열리는지
- 각 스테이지의 시간, 배경, 적 구성과 난이도가 다른지
- 보스 처치 후 프로토콜 카드 3개가 표시되는지
- 카드 밖을 눌러도 다음 스테이지로 넘어가지 않는지
- 프로토콜 효과가 기존 레벨업 강화와 함께 유지되는지
- 3·6·9스테이지 후 전투 등급이 상승하는지
- 10스테이지 격파 후 전체 승리로 끝나는지
- 후반 스테이지에서도 프레임이 유지되는지
