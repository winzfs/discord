# 14. 히어로 스트라이크 슈팅 모드

## 목적

기존 랜덤 디펜스와 퍼즐 모드에서 분리된 모바일 세로형 아케이드 슈팅게임입니다.

- 경로: `/hero-strike`
- 렌더링: React에서 호스팅하는 Canvas 2D
- 조작: 화면 상대 드래그 이동, 기본 공격 자동 발사
- 구성: 6개 독립 스테이지와 스테이지별 보스전
- 성장: 경험치 레벨업과 3택 영구 강화
- 저장: 브라우저 `localStorage` 최고 점수

## 핵심 플레이

1. 화면을 끌어 트레이서를 이동합니다.
2. 기본 공격은 자동 발사됩니다.
3. 적이 떨어뜨린 경험치 조각은 짧게 퍼진 뒤 아래로 가속됩니다.
4. 경험치 조각이 자석 범위에 들어오면 플레이어를 향해 끌려옵니다.
5. 레벨업 때 세 가지 강화 중 하나를 선택합니다.
6. 강화 카드는 현재 단계, 다음 단계, 최대 단계를 표시합니다.
7. 각 스테이지 제한 시간이 지나면 보스가 등장합니다.
8. 보스를 격파하면 클리어 화면에서 보상을 받고 다음 지역으로 이동합니다.
9. 6스테이지 보스를 격파하면 전체 승리합니다.

## 스테이지 구성

일반전은 45초부터 70초까지이며 보스전 시간은 별도입니다.

| 스테이지 | 일반전 | 일반 탄막 | 보스 패턴 |
|---|---:|---|---|
| KING'S ROW | 45초 | 단일 조준탄 | 7방향 부채꼴 |
| HANAMURA RUINS | 50초 | 좌우 분열탄 | 회전 꽃잎탄 |
| NULL SECTOR FACTORY | 55초 | 조준·십자탄 | 양방향 나선탄 |
| ANTARCTIC BASE | 60초 | 빗줄기 탄막 | 이동 레인 탄막 |
| JUNKERTOWN SKYWAY | 65초 | 넓은 부채꼴 | 고속 집중 연사 |
| GIBRALTAR ORBIT | 70초 | 고속 다중탄 | 집중·나선·레인 혼합 |

다음 스테이지 진입 시 체력 1, 보호막 1, 궁극기 기본 25%와 스테이지 보상 아이템을 받습니다. 펄스 드라이브가 있으면 스테이지 궁극기 보상도 증가합니다.

## 적 종류

- 러너: 빠르게 내려오며 직접 압박
- 드론: 기본 조준 사격
- 탱커: 높은 체력과 넓은 탄막
- 저격기: 상단에서 빠른 바늘탄 발사
- 지그재그 기체: 좌우로 움직이며 교차탄 발사
- 폭격기: 느리고 단단하며 대형 중탄 투하
- 보스: 스테이지별 전용 체력과 탄막 패턴 사용

## 경험치 조각 이동

- 생성 시 약하게 위와 옆으로 퍼집니다.
- 자석 범위 밖에서는 아래 방향으로 계속 가속됩니다.
- 낙하 속도에는 상한을 두어 화면을 지나치게 빠르게 벗어나지 않습니다.
- 경험치 조각 수명은 12초입니다.
- 낙하 중에는 빛 꼬리가 조각 위쪽에 표시됩니다.
- 자석 범위 안에서는 낙하보다 플레이어 흡수가 우선합니다.

## 레벨업 강화 공통 규칙

- 모든 강화는 재선택 시 실제 효과가 증가합니다.
- 카드 하단에 `LV.현재 → 다음 / 최대`가 표시됩니다.
- 카드 설명은 다음 단계에서 적용될 정확한 수치를 표시합니다.
- 최대 단계에 도달한 강화는 다음 선택지에서 제외됩니다.
- 첫 레벨업에는 유도 미사일이 반드시 포함됩니다.
- 이후에도 최소 한 개의 무기 계열 강화가 제공됩니다.

## 단계별 강화

### 기본 화력

- 속사 모듈: 단계마다 공격 간격 감소
- 펄스 확장: 정면 탄환 2발 → 3발 → 4발
- 고출력 코어: 모든 무기의 기준 공격력 누적 증가
- 관통 탄환: 단계와 같은 횟수만큼 기본탄 관통
- 오버클럭: 오버드라이브 피해 배율 단계별 증가
- 치명 코어: 치명타 확률과 피해 배율 동시 증가

### 보조 무기

- 유도 미사일
  - 발사 간격 감소
  - 속도와 회전 추적력 증가
  - 피해와 폭발 반경 증가
- 드론 편대
  - 연사 속도와 피해 증가
  - 4단계에서 관통 추가
- 측면 포대
  - 1단계 2발
  - 2단계 4발
  - 3단계 6발과 추가 관통
- 후방 방어포
  - 1단계 1발
  - 2단계 3발
  - 3단계 5발과 관통
- 폭발 탄두
  - 단계마다 폭발 반경과 추가 피해 비율 증가
- 연쇄 코어
  - 단계마다 전이 대상 수, 탐색 거리, 전이 피해 증가

### 생존과 편의

- 회수 자석: 단계마다 흡수 반경 증가 폭도 커짐
- 시간 방벽: 고단계에서 한 번에 더 많은 보호막과 더 긴 무적 시간 제공
- 펄스 드라이브
  - 선택 즉시 궁극기 충전량 증가
  - 처치, 그레이즈, 궁극기 아이템, 스테이지 보상의 획득량 영구 증가

## 필드 아이템

유도 미사일은 필드 아이템으로 등장하지 않고 레벨업 강화로만 획득합니다.

- 경험치 조각: 레벨업 경험치
- 회복 팩: 체력 1 회복
- 궁극기 코어: 궁극기 게이지 충전
- 방벽 코어: 보호막 1 획득
- 폭발 코어: 적탄 제거와 화면 전체 피해
- 오버드라이브 코어: 일정 시간 화력 강화
- 지원 드론 코어: 18초간 임시 드론 편대 강화
- 시간 왜곡 코어: 9초간 적 이동과 적탄 속도 감소
- 경험치 코어: 경험치 45 즉시 획득

## 피격 판정

- 트레이서 스프라이트 표시 크기: 72px 유지
- 실제 피격 반경: 9
- 캐릭터 중앙의 작은 청록색 원이 실제 피격점
- 아이템 수집 반경은 외형 기준으로 유지

## 시인성 원칙

- 적 기본탄: 분홍·빨강 원형과 흰 중심점
- 저격탄: 길고 뾰족한 바늘
- 교차탄: 마름모 파편
- 중탄: 큰 원형과 밝은 외곽선
- 경험치: 청록 다이아와 이동 반대 방향의 빛 꼬리
- 유도 미사일: 주황색 로켓과 추진 불꽃
- 드론 편대: 연두색 소형 기체
- 시간 감속: 청록색 원형 오라

색상만이 아니라 형태로도 위험물과 보상을 구분합니다.

## 모바일 성능 기준

- 탄환 최대 220개
- 유도 미사일 최대 16개
- 파티클 최대 140개
- 부유 텍스트 최대 10개
- 최대 1.5배 `devicePixelRatio` 렌더링
- 정적 배경 캐시
- 고비용 `shadowBlur` 미사용

## 배포 후 점검 항목

- 경험치가 짧게 퍼진 뒤 아래로 가속되는지
- 경험치 낙하 꼬리가 이동 방향과 반대로 표시되는지
- 강화 카드에 현재·다음·최대 단계가 표시되는지
- 같은 강화를 다시 선택했을 때 카드 수치와 실제 효과가 함께 증가하는지
- 측면 포대가 2·4·6발로 증가하는지
- 후방 방어포가 1·3·5발로 증가하는지
- 폭발탄의 반경과 피해가 함께 증가하는지
- 연쇄 코어의 대상 수·거리·피해가 함께 증가하는지
- 펄스 드라이브가 모든 궁극기 획득 경로에 적용되는지
- 스테이지 전환 후 영구 강화 단계가 유지되는지
- 후반 탄막에서도 프레임이 유지되는지
