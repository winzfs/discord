# 13. 유닛 소환/합성 연출 브리핑

## 1. 목적

유닛 소환, 도박 소환, 셀 3스택 합성, 신화 조합 시 결과 유닛이 어디에 생성되었는지 확실히 보이도록 셀 위치 기반 연출을 추가했습니다.

## 2. 적용 파일

```text
apps/web/src/game-client/pixi/pixiUnitRevealFxRuntime.ts
apps/web/src/game-client/pixi/pixiControlActionRuntime.ts
apps/web/src/game-client/pixi/pixiUnitActionRuntime.ts
```

## 3. 구현 내용

- `showUnitRevealFx()` 추가
- 결과 유닛의 보드 셀 중심 좌표를 계산해 이펙트를 표시합니다.
- 등급별 색상을 다르게 표시합니다.
- 원형 링, 빛 번짐, 방사형 라인, 텍스트가 함께 표시됩니다.
- 소환은 짧은 팝 연출로 표시합니다.
- 도박 소환은 획득 연출로 표시합니다.
- 셀 3스택 합성은 승급 연출로 표시합니다.
- 신화 조합은 신화 조합 전용 텍스트와 더 긴 연출로 표시합니다.

## 4. 등급별 표시

```text
일반: 흰색
희귀: 하늘색
영웅: 보라색
전설: 금색
신화: 분홍색
```

## 5. 연결된 액션

- 일반 소환 성공
- 도박 소환 성공
- 셀 3스택 합성 성공
- 신화 조합 성공

## 6. 확인 필요

```bash
pnpm typecheck
pnpm build:web
```

브라우저에서 확인할 항목:

- 일반 소환 시 생성 위치에 링/텍스트가 보이는지
- 희귀/영웅/전설 등급 색상이 구분되는지
- 도박 소환 성공 시 획득 연출이 보이는지
- 셀 3스택 합성 시 승급 연출이 보이는지
- 신화 조합 시 신화 조합 연출이 보이는지
- 모바일 화면에서 텍스트가 너무 작거나 셀 밖으로 과하게 튀지 않는지
