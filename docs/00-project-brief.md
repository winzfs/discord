# 00. 프로젝트 개요

## 1. 프로젝트 이름

가칭: **Dopamine Defense Web**

오버워치 디스코드 커뮤니티 서버를 위한 웹 대시보드와 랜덤 협동 디펜스 미니게임을 결합한 프로젝트입니다.

최종 이름은 서버 콘셉트에 맞춰 별도 결정합니다.

후보:

- 도파민 디펜스
- 랜덤 히어로 디펜스
- 듀오 웨이브 디펜스
- 서버 수호대
- 랜덤 파티 디펜스

## 2. 프로젝트 목표

이 프로젝트의 목표는 단순한 웹게임이 아니라, 디스코드 커뮤니티의 체류 시간과 참여 동기를 높이는 것입니다.

주요 목표:

1. 디스코드 계정으로 로그인한다.
2. 서버 멤버만 이용 가능한 웹 대시보드를 제공한다.
3. 짧고 가볍게 즐길 수 있는 랜덤 협동 디펜스 게임을 제공한다.
4. 게임 점수, 시즌 랭킹, 업적, 칭호를 디스코드 활동과 연결한다.
5. 서버 운영자가 이벤트, 랭킹, 보상을 관리할 수 있게 한다.

## 3. 참고 자료 활용 기준

업로드된 참고 문서는 다음 요소를 이해하기 위한 자료로만 사용합니다.

- 랜덤 소환형 디펜스의 기본 플레이 흐름
- 협동 웨이브 구조
- 유닛 등급 구조
- 유닛 합성/강화 구조
- 유물, 펫, 미션, 보상 같은 장기 성장 요소
- 일반 모드, 어려움 모드, 도전 모드 같은 모드 구분

다음 요소는 그대로 사용하지 않습니다.

- 원작 게임명
- 원작 캐릭터명
- 원작 스킬명
- 원작 유닛 이미지/아이콘/UI
- 원작 수치 테이블
- 원작 과금 구조
- 원작 고유 시스템명
- 원작의 세부 스테이지 구성

즉, 장르 문법은 참고하되 새 게임으로 기획합니다.

## 4. 목표 플랫폼

- PC 웹 브라우저
- 모바일 웹 브라우저
- 디스코드 인앱 브라우저
- 추후 PWA 확장 가능

초기 MVP에서는 별도 앱 설치 없이 웹 링크 접속만 지원합니다.

## 5. 핵심 사용자

### 5.1 일반 서버 멤버

- 디스코드 서버에서 놀거리를 찾는 유저
- 짧게 5~15분 정도 즐길 수 있는 게임을 원하는 유저
- 랭킹, 칭호, 프로필 꾸미기에 관심 있는 유저
- 친구와 같이 플레이하거나 점수를 비교하고 싶은 유저

### 5.2 서버 운영자

- 서버 활성도를 높이고 싶은 관리자
- 이벤트와 시즌 랭킹을 운영하고 싶은 관리자
- 서버 멤버의 활동 데이터를 한눈에 보고 싶은 관리자
- 디스코드 봇과 웹 대시보드를 함께 관리하고 싶은 관리자

## 6. MVP 범위

MVP는 반드시 작게 시작합니다.

### 포함

- 디스코드 OAuth 로그인
- 접속한 유저의 기본 프로필 표시
- 서버 멤버 여부 확인
- 간단한 대시보드 홈
- 싱글 또는 비동기 협동 느낌의 웨이브 디펜스
- 랜덤 유닛 소환
- 같은 등급 유닛 합성
- 웨이브 진행
- 전투 결과 저장
- 개인 최고 기록
- 서버 랭킹
- 기본 관리자 페이지 초안

### 제외

- 실시간 PvP
- 복잡한 매칭 시스템
- 실시간 2인 협동 네트워크 동기화
- 과금 시스템
- 모바일 앱 출시
- 복잡한 애니메이션
- 대규모 유닛 수집 시스템

실시간 협동은 Cloudflare Durable Objects 등을 이용해 추후 확장합니다.

## 7. 개발 원칙

1. 처음부터 너무 크게 만들지 않는다.
2. 큰 파일 하나에 모든 로직을 넣지 않는다.
3. 화면, 게임 엔진, 데이터, API, 인증, DB 접근을 분리한다.
4. 밸런스 데이터는 코드와 분리한다.
5. 디스코드 관련 로직은 별도 모듈로 격리한다.
6. Cloudflare 환경 변수와 비밀키는 문서화하되 저장소에 커밋하지 않는다.
7. 확장 가능성을 위해 폴더 구조를 먼저 정리한다.

## 8. 최종 서비스 이미지

유저는 디스코드 서버 공지 또는 봇 명령어를 통해 웹사이트에 접속합니다.

1. 디스코드 로그인
2. 서버 멤버 인증
3. 대시보드 홈 진입
4. 오늘의 랭킹, 내 프로필, 이벤트 확인
5. 랜덤 디펜스 게임 시작
6. 웨이브를 막으며 유닛 소환/합성/강화
7. 종료 후 점수 저장
8. 서버 랭킹 반영
9. 디스코드 채널에 시즌 기록 또는 이벤트 결과 공유

## 9. 성공 기준

MVP 성공 기준:

- 서버 멤버가 별도 설명 없이 로그인하고 게임을 시작할 수 있다.
- 한 판 플레이 시간이 5~10분 내외다.
- 결과가 저장되고 랭킹에 반영된다.
- 관리자 입장에서 시즌 랭킹을 확인할 수 있다.
- 이후 유닛, 유물, 펫, 모드 추가가 쉬운 구조다.
