# 20. 역할군별 유닛 밸런스 규칙

## 1. 목적

유닛 역할이 전투에서 명확히 구분되도록 기본 공격력과 스킬 효과 기준을 정리합니다.

핵심 방향:

```text
공격형: 직접 피해와 광역/연쇄 피해가 강함
제어형/탱커: 직접 피해는 낮고 감속/빙결/중력 제어가 강함
지원형/버프: 직접 피해는 낮고 주변/전역 버프, 경제, 보조 효과가 강함
```

## 2. 적용 파일

```text
packages/game/src/data/heroes.ts
apps/web/src/game-client/pixi/pixiCombatRuntime.ts
apps/web/src/game-client/pixi/pixiBaseHeroSkillRuntime.ts
```

## 3. 기본 공격력 기준

일반~전설 유닛의 `power`를 역할에 맞게 다시 조정했습니다.

### 3.1 공격형

공격형은 같은 등급 내에서 가장 높은 `power`를 가집니다.

목표:

- 보스 또는 선두 적을 빠르게 처치
- 광역/연쇄/관통 스킬 피해를 크게 체감
- 버프 유닛 없이도 최소한의 딜 역할 수행

예시:

| 등급 | 유닛 | 조정 후 power |
|---|---|---:|
| 일반 | 오버워치 신병 | 92 |
| 일반 | 정커 고철총잡이 | 98 |
| 희귀 | 탈론 추적요원 | 170 |
| 희귀 | 헬릭스 로켓 정찰병 | 160 |
| 영웅 | 비슈카르 광자 설계사 | 300 |
| 영웅 | 눌 섹터 전류 지휘체 | 320 |
| 전설 | 누미니 레일건 에이스 | 620 |
| 전설 | 감시위성 저격관 | 650 |

### 3.2 제어형/탱커

제어형은 공격형보다 낮은 `power`를 가집니다.

목표:

- 몬스터를 늦춰 공격형이 딜할 시간을 벌어줌
- 방벽/감속/빙결/중력 제어 체감 강화
- 직접 딜러처럼 보이지 않게 함

예시:

| 등급 | 유닛 | 조정 후 power |
|---|---|---:|
| 일반 | 헬릭스 경비병 | 48 |
| 일반 | 눌 섹터 감속 옴닉 | 42 |
| 희귀 | 십자군 수습대원 | 82 |
| 희귀 | 남극 연구기지 대원 | 74 |
| 영웅 | 헬릭스 전술 방패병 | 132 |
| 영웅 | 달 기지 중력 기술자 | 126 |
| 전설 | 옴닉 요새 집행기 | 260 |
| 전설 | 이지스 작전지휘관 | 240 |

### 3.3 지원형/버프형

지원형은 가장 낮은 직접 공격력을 가집니다.

목표:

- 자기 딜로 적을 잡기보다 주변/전역 버프 제공
- 경제 보너스와 공격 템포 보조
- 배치 전략을 요구하게 함

예시:

| 등급 | 유닛 | 조정 후 power |
|---|---|---:|
| 일반 | 오버워치 의무요원 | 34 |
| 일반 | 볼스카야 정비공 | 30 |
| 희귀 | 오버워치 야전 의무관 | 58 |
| 희귀 | 카이로 나노 보조병 | 52 |
| 영웅 | 오아시스 가속 기술자 | 88 |
| 영웅 | 정커 스크랩 정비공 | 96 |
| 전설 | 탈론 크레딧 해커 | 160 |
| 전설 | 오아시스 시간 예측관 | 170 |

## 4. 스킬 효과 기준

`pixiBaseHeroSkillRuntime.ts`에서 스킬 효과도 역할에 따라 다르게 적용합니다.

### 4.1 공격형 스킬

공격형은 다음 효과가 강합니다.

- `attack`
- `boss-killer`
- `area-damage`
- `burst`
- `chain`
- `multi-hit`
- `extra-hit`
- `pierce`
- `beam`

공격형은 스킬 피해 배율이 더 높게 적용됩니다.

### 4.2 제어형 스킬

제어형/탱커는 다음 효과가 강합니다.

- `slow`
- `freeze`
- `grouping`
- `debuff`

제어형은 직접 피해 대신 감속/빙결/중력 효과가 더 강하게 적용됩니다.

### 4.3 지원형 스킬

지원형은 다음 효과가 강합니다.

- `buff`
- `haste`
- `power-up`
- `team-wide`
- `economy`
- `coin-bonus`
- `wave-reward`

지원형은 자기 공격 피해 증가를 거의 받지 않으며, 대신 연계 버프와 경제 보너스가 역할의 핵심입니다.

## 5. 전투 런타임 기준

`pixiCombatRuntime.ts`의 fallback 배율도 역할별로 분리했습니다.

```text
공격형 fallback 피해 배율: 1.35
제어형/탱커 fallback 피해 배율: 0.62
지원형 fallback 피해 배율: 0.48
```

탱커 기본 공격에는 감속 효과를 더 강하게 적용합니다.

```text
기존: speed * 0.92
변경: speed * 0.88
```

## 6. 밸런스 의도

이번 조정 후 기대되는 플레이는 다음과 같습니다.

- 공격형 유닛 없이 지원/제어만 모으면 몬스터를 잡기 어렵습니다.
- 지원형 유닛은 딜러 옆에 배치해야 가치가 커집니다.
- 제어형 유닛은 직접 처치보다 시간을 버는 역할을 합니다.
- 광역 딜러와 단일 보스 딜러를 구분해서 배치해야 합니다.
- 전설 지원형은 강하지만 직접 딜러를 대체하지 못합니다.

## 7. 다음 조정 후보

- 유닛 정보창에 `공격형`, `제어형`, `버프형` 배지 표시
- 공격형은 빨간/노란 테두리, 제어형은 파란 테두리, 지원형은 초록 테두리 적용
- 세력 시너지와 역할 시너지를 분리
- 신화 유닛도 역할별 고유 수치를 다시 정리
- 버프 수치를 정보창에 숫자로 표시
