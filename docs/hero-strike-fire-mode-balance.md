# 히어로 스트라이크 DRIVE / FOCUS 발사 밸런스

## 목적

DRIVE와 FOCUS는 서로 다른 화력 단계가 아니라 같은 주무기를 운용하는 두 전투 자세입니다.

- DRIVE: 이동 자유도와 넓은 대응 범위
- FOCUS: 락온, 집탄, 약점과 BREAK 압박
- 기본 발사량과 지속 화력은 두 자세에서 크게 달라지지 않습니다.

## 공통 규칙

- FOCUS 진입 지연: 0.08초
- 기본 주무기 발사 간격 페널티 제거
- 기본 주무기 피해 격차는 무기 내부 계수를 포함해 약 10~20% 범위로 제한
- 보조무기 DRIVE 피해 감소는 5%, 발사 간격 증가는 3% 수준으로 제한
- 모드 특화 핵심 모듈을 선택했을 때만 해당 자세가 추가 강화됩니다.

## 펄스 리피터

- DRIVE와 FOCUS 모두 기본 5점사
- 사이클과 냉각 속도 동일
- FOCUS는 락온과 명중 충격이 우수
- 이론 지속 화력: FOCUS가 DRIVE보다 약 19% 높음

## 브리처

- DRIVE: 기본 5펠릿, 넓은 확산
- FOCUS: 기본 6펠릿, 좁은 집탄과 높은 BREAK 압력
- 펌프와 재장전 주기는 동일
- 이론 지속 화력: FOCUS가 DRIVE보다 약 9% 높음

## 아크 레일

- DRIVE: 축전하면서 경량 관통탄을 지속 발사
- FOCUS 진입: 저장된 축전량을 소비해 강한 관통 방출
- FOCUS 유지: 정밀 관통탄을 지속 발사
- DRIVE로 돌아가면 다음 강공격을 다시 축전
- 기본 지속탄 화력: FOCUS가 DRIVE보다 약 10% 높음

## 밸런스 원칙

- 이동을 강제하기 위해 화력을 박탈하지 않습니다.
- 멈춤을 보상하기 위해 탄환 수를 과도하게 늘리지 않습니다.
- FOCUS의 주된 보상은 정확도와 우선 표적 처리입니다.
- DRIVE의 주된 보상은 생존과 위치 조정입니다.
- 실제 모바일 플레이에서 모드별 체류 시간과 명중률을 확인한 뒤 5% 단위로만 조정합니다.
