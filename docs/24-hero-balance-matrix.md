# 24. 전체 영웅 밸런스표

## 1. 목적

영웅별 포지션, 공격타입, 스킬 발동 기준, 실제 역할을 한눈에 보기 위한 기준표입니다.

이 문서는 감으로 수치를 고치지 않고, 이후 밸런스 조정의 기준으로 사용합니다.

## 2. 공통 발동 기준

```text
공격 / 피해 / 광역 / 연쇄 계열: 기본 공격 시 42% 확률
제어 / 표식 / 취약 계열: 기본 공격 시 30% 확률
지원 / 경제 / 소환 / 방어 계열: 기본 공격 시 24% 확률
신화 궁극기: 궁극기 게이지 100%
자리야 입자포: 확률 스킬이 아니라 지속 강화 공격타입
```

## 5. 현재 밸런스 관찰

### 5.1 딜러

딜러는 대체로 전투력이 높고, 탱커/지원보다 직접 피해 기대값이 높습니다.

주의할 점:

```text
전설 딜러의 기본 전투력이 매우 높음
신화 캐서디의 전투력 760은 보스 대응 캐릭터로 적절하지만, 일반 웨이브까지 너무 잘 잡는지 확인 필요
트레이서는 공속이 매우 높으므로 42% 스킬 체감 발동 빈도가 높음
```

### 5.2 탱커

탱커는 전투력 자체는 낮거나 중간이지만, 기본 공격 시 감속이 있고 제어 스킬을 가집니다.

주의할 점:

```text
탱커 기본 감속 + 제어 스킬이 겹치면 러시 웨이브가 과하게 쉬워질 수 있음
자리야는 지속 강화가 있어서 장기전에서 기대값이 높음
```

### 5.3 지원

지원은 직접 피해보다 팀 연계, 경제, 공속, 증폭을 담당합니다.

주의할 점:

```text
지원 영웅이 많을수록 직접 화력이 낮아져야 하지만, 전역 지원이 누적되면 후반 기대값이 높아질 수 있음
경제 효과는 초반에 너무 강하면 스노우볼이 커짐
```
