# 22. 히어로 스트라이크 주무기 시스템 개편

## 1. 목표

주무기를 단순 DPS 수치가 아니라 서로 다른 조작 리듬과 자원 관리로 구분합니다.

## 2. 공통 원칙

- 주무기마다 독립 런타임을 사용합니다.
- 공통 `fireCooldown` 하나로 모든 무기를 제어하지 않습니다.
- DRIVE와 FOCUS는 무기마다 다른 의미를 가집니다.
- 전역 DRIVE/FOCUS 피해 보정으로 무기 차이를 덮지 않습니다.
- 보조무기는 주무기의 기본 공격력을 그대로 상속하지 않습니다.
- 전용 HUD로 열, 탄창, 축전 상태를 표시합니다.
- 강한 사격은 소리, 화면 반동, 탄도, 명중 반응이 함께 바뀝니다.

## 3. 펄스 리피터

- DRIVE: 3점사 제압 사격
- FOCUS: 5점사 정밀 버스트
- 발사할수록 열이 상승
- 과열 시 강제 냉각
- DRIVE 이동 중 냉각 효율 증가
- 열이 높을수록 마지막 탄의 반동과 피해가 조금 증가
- 전용 진화 `펄스 스톰`: 점사와 연동한 측면 탄막

핵심 감각: 빠른 점사와 열 관리.

## 4. 브리처 산탄포

- 5발 탄창
- DRIVE: 넓은 2펠릿 견제
- FOCUS: 좁은 6펠릿 집중 산탄
- 발사 뒤 펌프 동작
- 탄창이 비면 자동 재장전
- 이동 중 재장전 속도 소폭 증가
- 전용 진화 `브리치 노바`: 마지막 탄을 강화 폭발탄으로 변환

핵심 감각: 가까이 붙어 강하게 쏘고, 마지막 탄과 재장전 타이밍을 설계.

## 5. 아크 레일

- DRIVE 중 축전
- DRIVE에서 약한 자동 사격 제거
- FOCUS 진입 시 축전량을 소비해 1회 관통포 발사
- 완충 사격은 폭이 넓고 BREAK 압력이 큼
- 발사 후 다시 이동해야 재충전
- 전용 진화 `썬더 라인`: 연쇄 코어와 치명 코어를 관통포에 집중

핵심 감각: 회피하면서 충전하고, 정렬 후 한 발로 관통.

## 6. HUD

- 펄스: HEAT 게이지와 과열 경고
- 산탄: SHELL 5칸과 재장전 진행
- 레일: CAPACITOR 게이지와 READY 표시

## 7. 성장 방향

범용 강화 이름을 유지하되, 실제 효과는 현재 주무기의 행동 규칙을 바꿉니다.

- 사이클 튜닝: 펄스 점사/냉각, 산탄 펌프/재장전, 레일 축전 속도
- 발사 구조 확장: 펄스 버스트 탄 수, 산탄 펠릿 수, 레일 보조 광선
- 관통 탄환: 주무기 관통 횟수
- 고출력 코어: 주무기 기본 피해와 보조무기 기준 피해를 서로 다른 비율로 강화

진화는 선택한 주무기에 귀속합니다. 다른 주무기를 들고 전용 진화 조건만 맞추는 우회는 허용하지 않습니다.

## 8. 구현 원칙

- `heroStrikePrimaryWeaponSystem.ts`: 무기 상태와 발사 준비
- `heroStrikePrimaryWeaponProfiles.ts`: 무기별 기본 규칙과 튜닝 변환
- `heroStrikePrimaryWeaponRenderer.ts`: 무기별 월드 표현과 HUD
- `heroStrikePlayerFire.ts`: 주무기·사이클 연동 보조탄 생성
- `heroStrikeDamageRuntime.ts`: 피해, 폭발, 연쇄, 처치, 충돌
- `heroStrikeSupportBalance.ts`: 주무기와 독립된 보조무기 기준 피해
- `heroStrikeWeaponRuntime.ts`: 기존 호출부를 유지하는 얇은 진입점
