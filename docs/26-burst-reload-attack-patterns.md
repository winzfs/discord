# 26. 연타 / 재장전형 공격 패턴 정리

## 1. 목적

일부 영웅의 공격이 실제 캐릭터 컨셉과 다르게 한 번의 큰 타격처럼 보이는 문제를 개선했습니다.

광선형, 전기형, 권총/라이플형 공격은 다음 구조가 더 자연스럽습니다.

```text
약 100ms 간격으로 빠르게 여러 번 타격
→ 일정 시간 동안 연타
→ 잠깐 재장전/회복 시간
→ 다시 연타
```

## 2. 적용 방향

총 피해량은 크게 올리지 않고, 기존 1회 피해를 여러 타격으로 나눕니다.

목표:

```text
타격감은 더 빠르고 선명하게
밸런스는 기존 총 피해량에 가깝게 유지
과한 DPS 상승 방지
연타 사이클이 겹치지 않도록 재장전 간격 확보
```

## 3. 공통 기준

```text
광선/전기형: 약 520ms 동안 5타
권총/연사형: 약 90~100ms 간격
총기형 기본 공격: 약 100ms 간격 3~5타
단발/저격/레일건/강타형: 묵직한 1타 유지
```

## 4. 적용 영웅

### 4.1 자리야

```text
기존: 빔 연출 후 마지막에 1회 피해
1차 변경: 빔이 유지되는 동안 3회 펄스 피해
현재 변경: 약 520ms 동안 5회 펄스 피해
```

추가 조정:

```text
같은 대상 연속 공격 시 차지 증가 유지
첫 펄스에만 탱커 감속 적용
총 피해량은 기존 빔 피해를 5타로 분할
공격 간격에 재장전형 최소 간격 적용
```

### 4.2 윈스턴

```text
기존: 전기 빔 연출 후 마지막에 1회 피해
1차 변경: 전기 빔 중 3회 감전 피해
현재 변경: 약 520ms 동안 5회 감전 피해
```

추가 조정:

```text
주 대상/연쇄 대상 피해를 각각 5타로 분할
감속은 첫 타격 때만 적용
공격 간격에 재장전형 최소 간격 적용
```

### 4.3 트레이서

```text
기존: 6발 연사처럼 보이지만 마지막에 1회 피해
변경: 약 95ms 간격으로 6발 각각 실제 피해 적용
```

추가 조정:

```text
총 피해량은 기존 공격 피해를 6발로 분할
공격 간격에 재장전형 최소 간격 적용
```

### 4.4 D.Va

```text
기존: 산탄처럼 보이지만 마지막에 1회 피해
1차 변경: 산탄 공격을 4회 피해로 분할
현재 변경: 약 100ms 간격으로 5타 산탄 피해
```

추가 조정:

```text
총 피해량은 기존 공격 피해를 5타로 분할
공격 간격에 재장전형 최소 간격 적용
```

## 5. 일반 총기형 기본 공격

고유 FX가 없는 총기형 영웅은 기본 투사체를 여러 발로 나눕니다.

적용 대상:

```text
오버워치 신병: 5연사, 100ms 간격
정커 고철총잡이: 3연사, 100ms 간격
탈론 추적요원: 4연사, 100ms 간격
헬릭스 로켓 정찰병: 3연사, 100ms 간격
오버워치 야전 의무관: 4연사, 100ms 간격
```

저격/레일건/강타형 공격은 묵직한 단발감을 유지합니다.

## 6. 적용 파일

```text
apps/web/src/game-client/pixi/pixiCombatRuntime.ts
apps/web/src/game-client/pixi/pixiWinstonBeamRuntime.ts
apps/web/src/game-client/pixi/pixiHeroAttackFxRuntime.ts
```

## 7. 확인 필요

```bash
pnpm typecheck
pnpm build:web
```

테스트 체크리스트:

- 자리야 빔이 약 100ms 간격 5타로 보이는지
- 자리야 차지 증가가 유지되는지
- 윈스턴 전기 빔이 약 100ms 간격 5타로 보이는지
- 트레이서가 약 95ms 간격 6발 피해를 넣는지
- D.Va가 약 100ms 간격 5타 산탄 피해를 넣는지
- 총기형 기본 영웅들이 100ms 간격 3~5연사로 보이는지
- 총 피해량이 기존 대비 과하게 증가하지 않는지
- 재장전형 영웅들이 공격을 너무 자주 겹쳐서 하지 않는지
