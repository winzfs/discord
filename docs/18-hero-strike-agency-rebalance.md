# 18. 히어로 스트라이크 플레이 개입 개편

## 문제 진단

- 첫 레벨업 요구 경험치가 38로 지나치게 낮아 초반부터 강화 선택이 연속 발생했습니다.
- 속사, 공격력, 탄환 수와 보조무기가 동시에 곱연산되어 적 체력 곡선을 빠르게 추월했습니다.
- 자동사격과 보조무기가 플레이어 이동 여부와 무관하게 최대 효율로 작동했습니다.
- 초반 적 이동과 탄막은 화면 아래 중앙에 머무는 플레이를 충분히 압박하지 못했습니다.
- 레벨업 15종, 스테이지 프로토콜 9회와 진화가 겹쳐 개별 선택의 의미가 희석됐습니다.

## 개편 원칙

1. 이동과 위치 선정이 화력에 직접 연결될 것
2. 적 한 기가 명확한 역할과 대응법을 가질 것
3. 레벨업은 자주 끊는 보상이 아니라 빌드 방향을 정하는 중요한 선택일 것
4. 자동 보조무기는 주무기를 대체하지 않을 것
5. 난이도는 탄환 수와 체력만 늘리는 방식보다 행동 요구를 늘릴 것

## 성장 속도

### 경험치

첫 레벨업 요구량을 38에서 98로 높이고 이후 요구량도 완만하게 증가시킵니다.

```text
LV2   98
LV3  128
LV4  161
LV5  196
LV6  232
```

- 최대 전투 레벨: 18
- 최대 레벨 이후 경험치는 점수로 변환
- 강화 선택 뒤 남은 경험치는 다음 요구량의 75%까지만 유지
- 한 번의 경험치 획득으로 강화 화면이 연속해서 열리지 않음
- 스테이지 보상에서 무료 경험치 코어 제거
- 경험치 코어와 특수 아이템 드롭률 축소

목표는 스테이지당 평균 1~2회, 전체 캠페인 약 12~16회의 레벨업입니다.

### 강화 단계

대부분의 강화 최대 단계를 2~3단계로 축소했습니다.

- 속사: 최대 3단계, 최종 공격 간격은 시작값의 약 72% 이상
- 정면탄: 최대 2단계
- 공격력: 최대 3단계, 단계당 16%
- 관통: 최대 2단계
- 미사일·드론: 최대 3단계
- 폭발·연쇄: 최대 2단계
- 치명타: 최대 3단계
- 후방 자동포: 일반 레벨업 풀에서 제거

폭발탄과 오버클럭은 3스테이지, 연쇄 코어는 4스테이지부터 등장합니다.

### 프로토콜

모든 스테이지마다 선택하던 프로토콜을 2·4·6·8스테이지 클리어 후 네 번만 제공합니다.

- 최대 단계 3 → 2
- 공격력 프로토콜: 단계당 10% → 7%
- 연사 프로토콜: 단계당 7% → 4%
- 치명타 보너스 축소
- 블링크 최대 충전은 3회로 제한

## TEMPO

플레이어가 실제로 이동하면 TEMPO가 오르고, 오래 정지하면 감소합니다.

### 화력 배율

- 주무기: TEMPO 0에서 68%, TEMPO 100에서 110%
- 보조무기 피해: TEMPO 0에서 25%, TEMPO 100에서 100%
- 보조무기 발사 간격: TEMPO 0에서 2.5배, TEMPO 100에서 정상
- FLOW 획득량: TEMPO 0에서 50%, TEMPO 100에서 115%

블링크는 TEMPO 32를 즉시 제공하며, 그레이즈도 TEMPO를 소량 회복합니다. 피격 시 TEMPO 34가 감소합니다.

## 조준 포격

1.75초 이상 거의 움직이지 않으면 현재 위치에 포격 조준이 시작됩니다.

- 예고 시간: 0.95초
- 조준 시작 뒤 위치가 고정되어 이동으로 회피 가능
- 폭발 반경: 46
- 재사용 대기시간: 5.4초
- 스테이지 시작 시 포격 상태 초기화

정지 상태는 화력 손실뿐 아니라 직접적인 피격 위험을 만듭니다.

## 적 역할

### 러너

- 화면 진입 후 플레이어의 현재 위치를 조준
- 0.52초 경고 후 직선 돌진
- 한 위치에 머물면 충돌 경로에 직접 들어옴

### 탱커

- 정면 중앙에 방패 표시
- 레일을 제외한 정면 중앙 탄환 피해 70% 감소
- 좌우로 이동해 측면을 맞히거나 레일로 관통해야 효율적

### 저격기·위버·폭격기

- 저격기는 플레이어 현재 위치를 빠르게 압박
- 위버는 교차탄과 좌우 이동으로 정렬을 흔듦
- 폭격기는 넓은 영역을 비우게 하는 중탄을 사용

### 적 밀도

- 화면 적 상한을 기존 15~24기에서 10~16기로 축소
- 적 개별 체력과 역할은 강화
- 편대는 남은 적 상한만큼만 생성
- 많은 적을 자동 삭제하는 전투 대신 위협 우선순위를 판단하는 전투를 목표로 함

## 보스 약점

보스 좌우를 왕복하는 금색 약점이 표시됩니다.

- 약점 명중 피해: 1.24배
- 약점 BREAK 압력: 1.65배
- 일반 부위 BREAK 압력: 0.72배
- 레일 중앙탄은 약점과 BREAK 공략에 유리

플레이어가 보스 약점과 같은 X축으로 직접 이동해야 최대 효율이 나옵니다.

## 보조무기 제한

- 미사일과 드론 기본 피해·연사 하향
- TEMPO 배율을 피해와 재사용 시간에 모두 적용
- 진화 보너스 하향
- 드론 연쇄 대상 제한
- 미사일의 보스 피해와 BREAK 압력 감소

보조무기의 목표 피해 기여도는 전체의 약 20~35%입니다.

## 진화

새 최대 단계에 맞춰 조건을 재설정했습니다.

- 펄스 스톰: 속사 3 + 정면탄 2
- 헌터 스웜: 미사일 3 + 폭발 2
- 아크 오버로드: 연쇄 2 + 치명 3
- 이지스 윙: 드론 3 + 방벽 2

조건을 일찍 완성해도 5스테이지부터만 진화할 수 있습니다. 스테이지 전환 때 조건을 다시 검사해 누락을 방지합니다.

## 검증 기준

- 1스테이지 레벨업이 대체로 1~2회 범위인지
- 3초 이상 정지하면 TEMPO 손실과 포격 압박이 체감되는지
- TEMPO가 낮을 때 보조무기만으로 웨이브가 정리되지 않는지
- 러너 돌진, 탱커 방패와 보스 약점이 실제 이동을 요구하는지
- 보스 약점과 일반 부위의 BREAK 속도가 구분되는지
- 프로토콜 선택이 전체 런에서 네 번만 발생하는지
- TypeScript 검사와 프로덕션 웹 빌드가 통과하는지
