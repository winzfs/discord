# 18. 히어로 스트라이크 플레이 개입 개편

## 문제 진단

- 첫 레벨업 요구 경험치가 지나치게 낮아 초반부터 강화 선택이 연속 발생합니다.
- 속사, 공격력, 탄환 수와 보조무기가 동시에 곱연산되어 적 체력 곡선을 빠르게 추월합니다.
- 자동사격과 보조무기가 플레이어 이동 여부와 무관하게 최대 효율로 작동합니다.
- 초반 적 이동과 탄막은 화면 아래 중앙에 머무는 플레이를 충분히 압박하지 못합니다.
- 한 판의 강화 횟수가 너무 많아 개별 선택의 의미가 희석됩니다.

## 개편 원칙

1. 이동과 위치 선정이 화력에 직접 연결될 것
2. 적 한 기가 명확한 역할과 대응법을 가질 것
3. 레벨업은 자주 끊는 보상이 아니라 빌드 방향을 정하는 중요한 선택일 것
4. 자동 보조무기는 주무기를 대체하지 않을 것
5. 난이도는 탄환 수와 체력만 늘리는 방식보다 행동 요구를 늘릴 것

## 목표 수치

- 첫 레벨업: 1스테이지 중반 이후
- 스테이지당 레벨업: 평균 1~2회
- 전체 10스테이지 런: 약 12~16회
- 주무기 최종 화력: 초기 대비 약 3~4배 범위
- 보조무기 총 기여도: 전체 피해의 약 20~35%
- 정지 상태 장기 유지: 화력 손실과 조준 포격 위험 발생

## 핵심 시스템

### TEMPO

플레이어가 실제로 이동하면 TEMPO가 오르고, 오래 정지하면 감소합니다.

- 높은 TEMPO: 주무기 소폭 강화, 보조무기 정상 작동, FLOW 획득 보너스
- 낮은 TEMPO: 주무기 피해 감소, 보조무기 연사와 피해 감소
- 블링크 사용 시 즉시 TEMPO 획득
- 피격 시 TEMPO 감소

### 조준 포격

일정 시간 같은 위치에 머물면 현재 위치를 기준으로 경고 원이 나타납니다.

- 충분한 예고 시간 제공
- 포격 위치가 고정된 뒤 이동하면 회피 가능
- 명중 시 피해와 FLOW/TEMPO 손실
- 연속 발생하지 않도록 재사용 대기시간 적용

### 적 역할

- 러너: 플레이어의 마지막 위치를 향한 돌진
- 탱커: 정면 중앙 공격 피해 감소, 측면 정렬을 요구
- 저격기: 정지한 플레이어를 우선 조준
- 위버: 좌우 이동을 유도하는 교차 사격
- 폭격기: 넓은 영역을 비우게 만드는 탄막

### 보스 약점

보스 좌우에 이동하는 약점이 표시됩니다.

- 약점 명중 시 추가 피해와 BREAK 압력
- 약점은 일정 시간마다 좌우 이동
- 중앙에서 무작정 자동사격하는 것보다 위치를 맞추는 플레이가 유리

## 성장 개편

- 강화 최대 단계는 대부분 2~3단계로 축소
- 단계당 증가량은 명확하지만 곱연산 폭주는 제한
- 경험치 요구량과 적 경험치 보상을 재조정
- 한 번에 여러 레벨이 연속 발생하지 않도록 선택 후 전투 복귀
- 후방 자동 공격처럼 플레이 개입을 줄이는 강화는 축소
- 진화는 중후반에 완성되도록 스테이지 조건을 추가

## 검증 기준

- 1스테이지에서 레벨업이 2회를 크게 넘지 않는지
- 10초 이상 정지 플레이가 안전하고 효율적인 전략이 아닌지
- 보조무기만으로 일반 웨이브가 자동 정리되지 않는지
- 러너 돌진, 탱커 정면 방어와 보스 약점이 실제 이동을 요구하는지
- TypeScript 검사와 프로덕션 웹 빌드가 통과하는지
