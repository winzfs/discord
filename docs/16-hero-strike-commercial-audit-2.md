# 16. 히어로 스트라이크 상용 품질 2차 감사

## 감사 목적

히어로 스트라이크의 전체 전투, 성장, 보상, 조작과 모바일 UI를 처음부터 다시 점검했습니다.

이번 감사는 기능 수를 늘리는 작업이 아니라 다음 질문을 기준으로 시스템을 정리하는 작업입니다.

- 첫 60초 안에 무기와 조작의 차이가 느껴지는가
- 플레이어가 현재 빌드의 핵심을 설명할 수 있는가
- 어려움이 화면 혼잡이나 긴 플레이 시간에서 나오지 않는가
- 피격과 실패가 납득 가능한가
- 한 판이 끝났을 때 다시 시도할 이유가 있는가
- 모바일에서 필요한 정보만 즉시 읽히는가

## 핵심 플레이 기둥

1. **조작 숙련**: 상대 드래그 이동, 작은 피격점, 그레이즈, 블링크 회피
2. **빌드 숙련**: 출격 세팅, 빌드 가중 레벨업, 재추첨, 무기 진화
3. **전투 숙련**: 편대 공세, 엘리트 특성, 보스 3페이즈
4. **반복 동기**: 스테이지 목표, 런 등급, 연구 데이터

## 제거한 요소

### 자동 전투 등급

3·6·9스테이지에서 선택과 관계없이 공격력, 연사, 체력과 보호막을 지급하던 전투 등급을 제거했습니다.

제거 이유:

- 작전 프로토콜과 역할이 겹쳤습니다.
- 플레이어의 선택이 아닌 자동 성장이라 빌드 정체성을 흐렸습니다.
- 장기 런의 체력과 HUD 크기를 불필요하게 키웠습니다.

### 임시 지원 드론 아이템

영구 드론 편대 강화와 같은 무기를 잠시 제공하던 아이템을 제거했습니다.

제거 이유:

- 현재 드론 레벨과 임시 드론 상태가 HUD에서 충돌했습니다.
- 출격 보조 무기 선택의 의미를 약화했습니다.

### 시간 감속 아이템

적과 적탄만 느리게 하던 시간 감속 아이템을 제거했습니다.

제거 이유:

- 패턴 학습보다 일시적인 난이도 무효화에 가까웠습니다.
- 적 이동, 스폰, 탄환에 서로 다른 시간 배율을 적용해 전투 규칙을 복잡하게 만들었습니다.

### 저체감 작전 프로토콜

경험치, 자석 범위와 점수만 증가시키는 프로토콜을 제거했습니다.

남은 프로토콜은 체력, 화력, 치명타, 연사와 블링크처럼 전투 감각을 직접 바꾸는 다섯 종류입니다.

## 새 조작: 블링크

트레이서의 정체성을 살리는 능동 회피기입니다.

- 기본 충전 2회
- 현재 이동 목표 방향으로 96px 이동
- 방향 입력이 없으면 위쪽으로 이동
- 사용 중 0.55초 무적
- 도착 지점 주변 적탄 일부 제거
- 기본 재충전 5.5초
- 작전 프로토콜로 최대 4회와 재충전 3.8초까지 성장
- 스테이지 진입 시 1회 회복

블링크 사용 횟수는 결과 화면에 기록됩니다.

## 레벨업 드래프트 개선

기존 방식은 15개 강화 중 세 장을 거의 무작위로 제시했습니다.

새 드래프트는 다음 요소를 가중치로 사용합니다.

- 선택한 주무기와의 시너지
- 출격 때 선택한 보조 무기
- 이미 투자한 강화의 후속 단계
- 무기 진화에 필요한 짝 강화
- 현재 체력이 낮을 때의 생존 선택지
- 강화 희귀도와 현재 플레이어 레벨

한 번에 최소 한 개의 무기 계열 강화를 보장합니다.

### 재추첨

- 기본 1회
- 연구 등급 8 이상에서 1회 추가
- 스테이지 목표 세 번 달성할 때마다 1회 추가
- 현재 제시된 세 장을 제외하고 다시 추첨
- 사용 횟수는 결과 화면에 기록

## 편대 웨이브

일반 적이 한 기씩 무작위로 내려오는 반복감을 줄이기 위해 편대 디렉터를 추가했습니다.

편대 종류:

- V자 돌입
- 양측 협공
- 중앙 종대
- 창끝 돌파
- 중장갑 호위

편대는 약 8~12초마다 등장하며 스테이지, 웨이브와 난이도에 따라 간격이 달라집니다.

- 웨이브 전환 직후 편대 도착 시간이 짧아집니다.
- 엘리트 출현 직전과 직후에는 편대 간격을 확보합니다.
- 화면 적 수 상한에 가까우면 편대를 생성하지 않습니다.

## 캠페인 길이 조정

10개 스테이지는 유지하되 일반전 시간을 48~68초로 압축했습니다.

| 스테이지 | 일반전 |
|---:|---:|
| 1 | 48초 |
| 2 | 50초 |
| 3 | 52초 |
| 4 | 54초 |
| 5 | 56초 |
| 6 | 58초 |
| 7 | 60초 |
| 8 | 62초 |
| 9 | 64초 |
| 10 | 68초 |

일반전 합계는 9분 32초입니다. 보스와 선택 화면을 포함한 목표 런 길이는 약 12~15분입니다.

후반 난이도는 긴 제한 시간이 아니라 다음 요소로 증가합니다.

- 편대 빈도와 구성
- 엘리트 특성
- 적 조합
- 탄환 패턴
- 보스 페이즈

## 난이도와 안전장치

- 플레이어 표시 크기 72px, 실제 피격 반경 9 유지
- 피격 후 1.25초 무적
- 피격 지점 주변 적탄 정리
- 저체력일 때 적탄 상한과 스폰 압박 완화
- 적탄 최대 약 136발
- 보스 입장과 페이즈 전환 때 적탄 정리
- 블링크를 사용해 실수 전 회피 가능

## 스테이지 목표 보상

목표 달성 보상:

- 난이도 배율이 적용된 추가 점수
- 연구 데이터
- 보호막 1칸
- 누적 세 번째 목표마다 재추첨 1회

무피격으로 스테이지를 완료하면 `PERFECT` 통계에 기록됩니다.

## 런 결과 화면

한 판 종료 시 다음 정보를 표시합니다.

- 도달 스테이지와 런 등급
- 점수, 처치 수, 최대 콤보
- 누적 피해량
- 피격 횟수
- 블링크 사용 횟수
- 목표 달성 수
- 무피격 스테이지 수
- 재추첨 사용 횟수
- 완성한 진화 무기
- 획득 연구 데이터
- 사용한 출격 세팅

## 연구 보상 원칙

연구 데이터는 단순 생존 시간보다 숙련 플레이를 더 보상합니다.

- 도달 스테이지
- 점수
- 목표 달성
- 무피격 스테이지
- 무기 진화
- 전체 승리

영구 연구 공격력과 경험치 보너스는 기존 상한을 유지해 반복 플레이가 난이도를 완전히 무효화하지 않게 합니다.

## 성능 목표

- 적 최대 약 24기
- 적탄 최대 약 136발
- 전체 탄환 최대 220발
- 유도 미사일 최대 16발
- 파티클 최대 140개
- 렌더 배율 최대 1.5
- 고비용 그림자 대신 단순 발광 도형 사용

## 배포 전 검증 항목

- TypeScript 전체 검사
- 웹 프로덕션 빌드
- 삭제한 `combatRank`, `supportDroneTime`, `timeWarp` 참조 부재
- 블링크 버튼과 궁극기 버튼 터치 영역 분리
- 재추첨 후 같은 카드 세 장이 그대로 나오지 않는지
- 편대가 적 수 상한을 초과하지 않는지
- 모든 프로토콜 최대 단계 후 다음 스테이지가 멈추지 않는지
- 결과 연구 보상이 한 번만 저장되는지
