# 17. 버프 제공 유닛 설명 표기 규칙

## 1. 목적

전투 중 실제로 연계 버프를 제공하는 유닛은 유저가 설명만 보고도 역할을 알 수 있어야 합니다.

따라서 버프 제공 유닛의 `description`에는 다음 문구를 명확히 포함합니다.

```text
버프형 지원 유닛
주변 8칸 아군에게 연계 공격력 버프
전역 버프형 지원 유닛
팀 전체에 연계 공격력 버프
```

## 2. 현재 버프 판정 기준

현재 전투 시너지 런타임 기준으로 버프 제공 유닛은 다음 조건을 만족합니다.

```text
role === support
그리고 다음 태그 중 하나 보유:
buff / haste / power-up / attack-speed / team-wide
```

관련 파일:

```text
apps/web/src/game-client/pixi/pixiHeroSynergyRuntime.ts
```

## 3. 설명에 명시해야 하는 버프 유형

### 3.1 주변 8칸 연계 버프

주변 8칸에 있는 아군에게 공격력 연계 보너스를 주는 유닛입니다.

설명 예시:

```text
버프형 지원 유닛으로, 주변 8칸 아군에게 연계 공격력 버프를 제공한다.
```

현재 대상:

| 유닛 | 설명 표기 |
|---|---|
| 오버워치 의무요원 | 주변 8칸 아군에게 연계 공격력 버프 |
| 볼스카야 정비공 | 주변 8칸 아군에게 연계 공격력 버프 |
| 오버워치 야전 의무관 | 주변 8칸 아군에게 연계 공격력 버프 |
| 카이로 나노 보조병 | 주변 8칸 아군에게 연계 공격력 버프 |
| 오아시스 가속 기술자 | 주변 8칸 아군에게 연계 공격력 버프 |
| 아나 | 주변 8칸 아군에게 연계 공격력 버프 |
| 키리코 | 주변 8칸 아군에게 연계 공격력 버프 |

### 3.2 전역 연계 버프

보드 전체 아군에게 공격력 연계 보너스를 주는 유닛입니다.

설명 예시:

```text
전역 버프형 지원 유닛으로, 팀 전체에 연계 공격력 버프를 제공한다.
```

현재 대상:

| 유닛 | 설명 표기 |
|---|---|
| 탈론 크레딧 해커 | 팀 전체에 연계 공격력 버프 |
| 이지스 작전지휘관 | 팀 전체에 연계 공격력 버프 |
| 오아시스 시간 예측관 | 팀 전체에 연계 공격력 버프 |

## 4. 주의할 점

`support` 역할이라고 해서 전부 버프 유닛은 아닙니다.

예시:

- 정커 스크랩 정비공은 포탑/지원 화력 유닛입니다.
- 일리아리는 지원형 폭딜 유닛입니다.
- 이런 유닛은 실제 연계 버프 판정 대상이 아니면 설명에 버프 제공 문구를 넣지 않습니다.

## 5. 구현 반영 위치

설명 수정 위치:

```text
packages/game/src/data/heroes.ts
```

표시 위치:

```text
apps/web/src/pages/LobbyPage.tsx
apps/web/src/game-client/pixi/pixiUnitInfoView.ts
```

## 6. 이후 개선 후보

- 유닛 카드에 `버프` 배지를 별도로 표시
- 선택 시 주변 8칸 버프 범위와 실제 적용 대상 하이라이트
- 전역 버프 유닛은 별도 아이콘 표시
- 버프 수치를 설명에 자동 삽입
