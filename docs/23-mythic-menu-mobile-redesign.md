# 23. 신화 조합 메뉴 모바일 레이아웃 정리

## 1. 목적

모바일 화면에서 신화 조합 메뉴가 아래로 드래그되지 않고, 재료 라벨이 너무 산만하게 보이는 문제를 정리했습니다.

## 2. 적용 파일

```text
apps/web/src/game-client/pixi/pixiMythicMenuView.ts
```

## 3. 스크롤 수정

기존 구조에서는 각 조합 행이 `pointertap`을 직접 받아 스크롤 드래그와 충돌할 수 있었습니다.

변경 후:

- 스크롤 영역에 투명 hit area 추가
- 행의 `pointertap` 제거
- 조합 가능 행은 `pointerdown → pointermove 거리 체크 → pointerup` 방식으로 처리
- 드래그 거리가 일정 이상이면 조합 터치로 보지 않음
- 모바일에서는 목록 영역을 위아래로 드래그해 스크롤

## 4. 레이아웃 정리

기존 재료 칩은 항목마다 색 테두리와 상태 텍스트가 붙어 화면이 복잡했습니다.

변경 후:

```text
재료 2/4 · 부족 2
✓[일반] 헬릭스…  ✓[영웅] 헬릭스…
·[영웅] 정커…    ·[전설] 옴닉…
```

핵심 원칙:

- 큰 칩 나열 제거
- 보유/전체/부족 수량을 먼저 표시
- 재료 세부명은 2줄로 표시
- 재료 세부 글씨 크기를 기존보다 키움
- 각 재료 이름 앞에 `[일반]`, `[희귀]`, `[영웅]`, `[전설]`, `[신화]` 등급 라벨 표시
- 너무 긴 이름은 각 줄 안에서만 말줄임 처리
- 조합 가능 행만 배경을 살짝 강조

## 5. 테스트 체크리스트

- 신화 메뉴 목록을 아래로 드래그할 수 있는지 확인
- 조합 가능 행을 짧게 터치하면 조합되는지 확인
- 드래그 중 실수로 조합되지 않는지 확인
- 재료 라벨이 오른쪽으로 삐져나가지 않는지 확인
- 재료 라벨이 2줄로 읽기 좋게 나오는지 확인
- 각 재료 이름 앞에 등급 라벨이 붙는지 확인
- 하단 안내 문구가 목록을 가리지 않는지 확인
