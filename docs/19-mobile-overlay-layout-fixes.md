# 19. 모바일 오버레이 레이아웃 수정

## 1. 목적

모바일 화면에서 신화 조합 메뉴와 유닛 정보창이 겹치거나, 텍스트가 패널 밖으로 나가는 문제를 수정했습니다.

## 2. 신화 조합 메뉴 수정

적용 파일:

```text
apps/web/src/game-client/pixi/pixiMythicMenuView.ts
```

반영 내용:

- 메뉴 전체 높이를 화면 안에 들어오도록 제한했습니다.
- 조합 목록을 내부 스크롤 영역으로 분리했습니다.
- 모바일에서는 위아래 드래그로 조합표를 스크롤할 수 있습니다.
- PC/브라우저 환경에서는 휠 스크롤도 지원합니다.
- 조합 재료 텍스트를 칩 형태로 바꾸고, 한 줄로 밖으로 튀지 않게 최대 너비를 제한했습니다.
- 재료는 최대 2열로 배치되며, 긴 이름은 말줄임 처리합니다.
- 재료 칩 테두리 색상은 해당 유닛/등급 색상에 맞춥니다.
- 조합 대상 이름도 등급 색상으로 표시합니다.

## 3. 등급 색상 기준

현재 색상 기준:

```text
신화: 노란색
전설: 진한 주황색
영웅: 마젠타 보라색
희귀: 시안 블루
일반: 회색
```

희귀와 영웅 등급이 모바일에서 비슷하게 보이지 않도록 다음처럼 대비를 벌렸습니다.

```text
희귀: 0x29d9ff
영웅: 0xf04bff
```

## 4. 유닛 정보창 가림 수정

적용 파일:

```text
apps/web/src/game-client/pixi/pixiGameLayerOrder.ts
```

기존 레이어 순서에서는 유닛 정보창이 몬스터/공격 이펙트 레이어보다 아래에 있어, 몬스터가 지나가면 정보창 일부가 가려질 수 있었습니다.

변경 후 레이어 순서:

```text
world
board
effects
hud
controls
info
menuLayer
```

이제 유닛 정보창은 몬스터와 이펙트보다 위에 표시되고, 신화 조합 메뉴는 가장 위에 유지됩니다.

## 5. 테스트 체크리스트

- 신화 조합 메뉴가 화면 밖으로 넘치지 않는지 확인
- 조합표를 세로로 드래그해 아래 항목까지 볼 수 있는지 확인
- 재료 텍스트가 패널 오른쪽 밖으로 튀지 않는지 확인
- 보유 재료 칩이 노란색 계열로 표시되는지 확인
- 재료 등급에 따라 칩 테두리 색상이 다르게 표시되는지 확인
- 희귀 칩은 시안 블루, 영웅 칩은 마젠타 보라색으로 구분되는지 확인
- 유닛 정보창이 몬스터 아래로 가려지지 않는지 확인
- 신화 조합 메뉴는 유닛 정보창보다 위에 뜨는지 확인
