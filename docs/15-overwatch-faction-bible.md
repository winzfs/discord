# 15. 오버워치 세력 세계관 바이블

## 1. 문서 목적

이 문서는 오버워치 세계관의 주요 세력, 지역, 기술권, 갈등 구조를 팬게임용 랜덤 디펜스에 맞게 정리한 세력 바이블입니다.

중요 원칙:

- 이 프로젝트는 비상업 팬게임입니다.
- 오버워치 및 관련 명칭, 캐릭터, 설정은 Blizzard Entertainment의 자산입니다.
- 정사 기반 요소와 게임용 재해석 요소를 분리합니다.
- 실제 코드에서는 이름이 바뀌어도 유지보수 가능하도록 `id`, `displayName`, `role`, `skillIds`, `tags`를 분리합니다.
- 일반/희귀 유닛은 공식 영웅을 직접 늘리는 방식보다, 세계관 속 세력의 병과/요원/시민군/옴닉/용병을 팬게임용으로 재해석합니다.

## 2. 참고 기준과 해석 등급

### 2.1 정사 기반 요소

다음은 공식 영웅 배경, 맵 설정, 시네마틱, 단편 소설, 인게임 대사에서 반복적으로 언급되는 요소를 기준으로 삼습니다.

- 오버워치: 옴닉 사태 이후 세계 평화를 위해 활동했던 국제 태스크포스
- 탈론: 암살, 테러, 생체/군사 기술을 활용하는 비밀 조직
- 하시모토: 카네자카를 장악하려는 일본 범죄 조직
- 정커타운: 옴니움 폭발 이후 황무지화된 호주권의 생존자/약탈자 도시권
- 비슈카르: 하드라이트 기술과 도시 개발을 앞세운 거대 기업
- 헬릭스 시큐리티: 군사 보안 및 중요 시설 방어를 맡는 민간 군사/보안 조직
- 연구기지/과학권: 남극 생태 관측소, 달 기지, 고도 과학 도시, 옴닉 연구 시설 등

### 2.2 팬게임 재해석 요소

다음은 게임성을 위해 추가하는 재해석입니다.

- 각 세력에 일반/희귀/영웅/전설 병과 체계를 부여합니다.
- 공식 영웅이 아닌 하위 유닛을 만들어 랜덤 소환 풀을 풍성하게 합니다.
- 실제 정사에 없는 세부 병과명은 팬게임용 명칭으로 사용합니다.
- 세력별 전투 스타일을 디펜스 역할군으로 변환합니다.

## 3. 세력 설계 원칙

각 세력은 단순히 이름만 다른 유닛이 아니라, 다음 차이를 가져야 합니다.

| 설계 축 | 설명 |
|---|---|
| 이념 | 왜 싸우는가 |
| 기술 | 무엇을 무기로 쓰는가 |
| 병과 | 어떤 하위 유닛이 나오는가 |
| 갈등 | 내부/외부 충돌 구조 |
| 게임 역할 | 탱커/딜러/지원/제어/경제 중 어디에 강한가 |
| 조합성 | 어떤 상위 영웅이나 신화 조합 재료로 자연스러운가 |

## 4. 오버워치 계열

### 4.1 핵심 정체성

오버워치 계열은 재난 대응, 옴닉 사태 수습, 민간인 보호, 국제 구조 작전의 이미지를 가집니다. 단순한 군대라기보다, 과학자·파일럿·의무요원·특수요원이 섞인 국제 구조/전투 조직입니다.

### 4.2 전투 스타일

- 균형형 병과
- 초보자에게 안정적인 유닛 풀
- 딜러, 탱커, 지원을 고르게 보유
- 단일 성능은 극단적이지 않지만 조합 안정성이 높음
- 지원형 주변 버프와 계정 성장 시스템과 잘 어울림

### 4.3 내부 갈등

오버워치 계열 유닛은 정의롭지만 완전무결하지 않습니다.

- 해체 이후 재소집에 대한 의견 차이
- 오래된 지휘 체계와 새로운 세대의 충돌
- 민간 구호와 군사 개입 사이의 윤리 문제
- 탈론/눌 섹터와의 대립 속 민간 피해 부담

### 4.4 디펜스 게임 변환

| 병과 | 역할 | 게임 기능 |
|---|---|---|
| 오버워치 신병 | 딜러 | 빠른 단일 공격, 초반 안정 딜 |
| 야전 의무관 | 지원 | 주변 공격력/공격속도 보조 |
| 방벽 수습대원 | 탱커 | 감속, 지연, 코어 보호 |
| 파일럿 후보생 | 딜러 | 빠른 공격, 낮은 체력 적 마무리 |
| 구조 기술자 | 지원 | 경제 보조, 수리/보상 보너스 |

## 5. 탈론 계열

### 5.1 핵심 정체성

탈론은 세계 질서를 흔들어 강한 자가 살아남는 세계를 만들려는 비밀 군사 조직으로 해석합니다. 암살, 침투, 심리전, 생체 기술, 불법 무기 거래, 정밀 타격이 어울립니다.

### 5.2 전투 스타일

- 단일 대상 처치 특화
- 보스 킬러, 표식, 약점 노출
- 빠른 적 처치 후 연쇄 보너스
- 높은 공격력 대신 낮은 안정성
- 지원보다 딜러/암살자 쪽에 강함

### 5.3 내부 갈등

- 조직원마다 목적이 다름
- 과학자, 암살자, 용병, 정치 공작원이 느슨하게 얽힘
- 강함을 증명하려는 노선과 순수 이익을 따지는 노선 충돌
- 오버워치와의 대립뿐 아니라 내부 배신 가능성도 큼

### 5.4 디펜스 게임 변환

| 병과 | 역할 | 게임 기능 |
|---|---|---|
| 탈론 추적요원 | 딜러 | 보스 표식, 단일 피해 증가 |
| 탈론 감시병 | 딜러 | 낮은 체력 적 우선 공격 |
| 생체 실험병 | 지원 | 피해 증폭, 위험한 버프 |
| 암살 연락책 | 딜러 | 처치 시 추가 피해/연쇄 공격 |
| 정보 브로커 | 지원/경제 | 처치 보상 증가, 웨이브 정보 제공 |

## 6. 하시모토 / 카네자카 범죄권

### 6.1 핵심 정체성

하시모토는 카네자카를 장악하려는 조직폭력 세력으로 해석합니다. 시마다 가문 이후 생긴 권력 공백을 차지하려는 범죄 조직이며, 지역 상점, 거리, 신사, 뒷골목을 장악하려는 분위기가 중요합니다.

### 6.2 전투 스타일

- 근거리/중거리 기습
- 표식, 출혈, 추가타
- 지역 장악형 디버프
- 요괴/카네자카 자경단과 대립
- 희귀 이하 유닛은 뒷골목 병력, 상위 유닛은 간부/기계화 보스 후보

### 6.3 내부 갈등

- 옛 시마다식 명예와 새로운 하시모토식 공포 정치의 대비
- 주민을 지키는 요괴/카네자카 자경단과 대립
- 돈과 체면, 공포를 동시에 추구하는 조직 문화
- 기계화/사이버네틱 간부가 등장할 경우 탈론/비슈카르와도 거래 가능

### 6.4 디펜스 게임 변환

| 병과 | 역할 | 게임 기능 |
|---|---|---|
| 하시모토 말단검객 | 딜러 | 빠른 단일 공격 |
| 카네자카 감시꾼 | 제어 | 표식, 약점 노출 |
| 골목 투척병 | 딜러 | 작은 범위 피해 |
| 요괴 자경단원 | 지원/딜러 | 카네자카 방어 버프 |
| 하시모토 간부 | 전설 후보 | 표식 대상 폭발, 보스 약화 |

## 7. 정커타운 / 황무지권

### 7.1 핵심 정체성

정커타운은 질서보다 생존, 폭발물, 고철, 거래, 투기장이 지배하는 황무지 도시권입니다. 정커 계열은 체계적인 군대가 아니라, 각자의 무기와 미친 기술력으로 버티는 생존자 집단입니다.

### 7.2 전투 스타일

- 폭발, 도탄, 함정, 고철 포탑
- 안정성 낮지만 고점이 높음
- 범위 피해와 랜덤 추가 효과
- 이자/경제 시스템과도 연결 가능: 고철 재활용, 전리품 추가 획득

### 7.3 내부 갈등

- 여왕 중심의 질서와 자유분방한 약탈자 문화 충돌
- 투기장 챔피언, 고철 상인, 폭탄 제조자, 추방자 간 갈등
- 외부 세력의 무기 거래와 내부 권력 싸움

### 7.4 디펜스 게임 변환

| 병과 | 역할 | 게임 기능 |
|---|---|---|
| 정커 고철총잡이 | 딜러 | 도탄 추가타 |
| 고철 폭파수 | 딜러 | 범위 폭발 |
| 황무지 방패꾼 | 탱커 | 누수 감소, 몸빵 |
| 스크랩 정비공 | 지원/경제 | 처치 보상 증가 |
| 투기장 챔피언 | 전설 후보 | 주변 적이 많을수록 피해 증가 |

## 8. 비슈카르 계열

### 8.1 핵심 정체성

비슈카르는 질서, 도시 개발, 하드라이트, 통제된 아름다움을 앞세운 기업권력입니다. 겉으로는 도시 개선과 인프라를 말하지만, 실제 플레이 감성에서는 통제, 배치, 구조물, 보호막, 레이저가 중요합니다.

### 8.2 전투 스타일

- 하드라이트 구조물
- 보호막, 경로 통제, 느린 범위 제어
- 지원형 버프와 방어형 탱커에 강함
- 직접 화력보다는 판을 설계하는 타입

### 8.3 내부 갈등

- 도시 개발이라는 명분과 주민 통제의 충돌
- 기업 효율과 인간 삶의 존엄 사이의 긴장
- 탈론/하시모토와 직접 동맹은 아니지만, 계약과 이익에 따라 위험한 거래 가능

### 8.4 디펜스 게임 변환

| 병과 | 역할 | 게임 기능 |
|---|---|---|
| 비슈카르 설계요원 | 지원 | 주변 버프, 구조물 강화 |
| 하드라이트 감시병 | 탱커 | 방벽, 감속 |
| 광자 기술자 | 딜러 | 관통/빔 공격 |
| 도시 통제 드론 | 제어 | 적 이동 속도 감소 |
| 하드라이트 감독관 | 전설 후보 | 전역 방어/공격속도 보조 |

## 9. 헬릭스 코퍼레이션 / 군사 보안권

### 9.1 핵심 정체성

헬릭스 계열은 질서, 보안, 대테러, 중요 시설 방어를 담당하는 군사 보안 조직으로 해석합니다. 이들은 오버워치처럼 이상주의적이지 않고, 계약과 임무, 시설 방어를 우선합니다.

### 9.2 전투 스타일

- 안정적인 사거리와 방어
- 로켓, 방벽, 제압 사격
- 보스 웨이브 대비에 강함
- 초반 희귀 딜러/탱커 풀에 적합

### 9.3 내부 갈등

- 세계 평화보다 계약과 시설 방어를 중시
- 오버워치 잔당과 협력할 수 있지만 항상 신뢰하지는 않음
- 비슈카르와 계약 관계가 생길 수 있고, 탈론 침투에 취약할 수 있음

### 9.4 디펜스 게임 변환

| 병과 | 역할 | 게임 기능 |
|---|---|---|
| 헬릭스 경비병 | 탱커 | 초반 감속/지연 |
| 파라 분대 로켓병 | 딜러 | 범위 로켓 피해 |
| 보안 저격수 | 딜러 | 긴 사거리, 선두 적 우선 |
| 진압 드론 | 제어 | 짧은 감속, 표식 |
| 헬릭스 장교 | 지원 | 특정 행/열 화력 보정 |

## 10. 연구소 / 과학권

### 10.1 핵심 정체성

연구소 계열은 남극 관측, 달 기지, 오아시스식 고도 과학, 옴닉 연구, 생체 기술 등으로 나뉩니다. 연구소 유닛은 군인이 아니라 과학 장비를 전투에 끌고 온 느낌이 중요합니다.

### 10.2 전투 스타일

- 빙결, 중력, 전기, 생체 증폭
- 특수 상태이상과 보조 효과
- 직접 화력보다 제어와 시너지
- 희귀/영웅 등급에서 개성이 잘 살아남

### 10.3 내부 갈등

- 과학 발전과 군사화 사이의 긴장
- 연구 자료를 노리는 탈론/비슈카르/하시모토와 충돌
- 실험 실패, 시설 붕괴, 고립된 연구팀 서사 가능

### 10.4 디펜스 게임 변환

| 병과 | 역할 | 게임 기능 |
|---|---|---|
| 남극 연구기지 대원 | 탱커/제어 | 빙결, 감속 |
| 달 기지 기술자 | 딜러/제어 | 중력장, 밀집 제어 |
| 생체 연구 보조병 | 지원 | 피해 증폭/회복형 버프 |
| 옴닉 분석가 | 지원/경제 | 웨이브 정보, 보상 보정 |
| 중력 실험체 | 영웅 후보 | 광역 끌어당김 |

## 11. 눌 섹터 / 옴닉 세력

### 11.1 핵심 정체성

눌 섹터와 옴닉 계열은 오버워치 세계관의 핵심 갈등축인 인간과 옴닉의 관계를 다룹니다. 모두가 적대적인 것은 아니며, 민간 옴닉, 독립 옴닉, 급진 옴닉, 군사용 옴닉을 분리하는 것이 중요합니다.

### 11.2 전투 스타일

- 기계식 병력, 드론, 감속 오일, 전자전
- 낮은 등급에서는 감속/지연 유닛
- 상위 등급에서는 빔, 방벽, 복제, 군집 제어
- 인간 세력과 섞을 때 윤리적 긴장감 발생

### 11.3 내부 갈등

- 평화적 옴닉과 급진 옴닉의 갈등
- 인간 사회의 차별과 폭력에 대한 반발
- 탈론이 갈등을 이용할 가능성
- 오버워치가 무력 진압과 화해 사이에서 고민하는 구조

### 11.4 디펜스 게임 변환

| 눌 섹터 감속 옴닉 | 탱커/제어 | 감속, 지연 |
| 옴닉 보초병 | 딜러 | 안정 사격 |

## 14. 현재 일반/희귀 유닛 리테마 기준

일반/희귀 유닛은 공식 영웅의 복제물이 아니라, 세계관 속 조직 병과로 설계합니다.

|---|---|---|
| spark-runner | 오버워치 신병 | 오버워치 |
| rookie-guard | 헬릭스 경비병 | 헬릭스 |
| mini-mender | 오버워치 의무요원 | 오버워치 |
| scrap-gunner | 정커 고철총잡이 | 정커타운 |

| pulse-ranger | 탈론 추적요원 | 탈론 |

## 15. 앞으로의 확장 방향

- 각 세력마다 6~10개 유닛을 확보합니다.
- 세력 시너지 태그를 추가합니다.
- 보스 웨이브마다 세력 침공 테마를 붙입니다.
- 패스 보상에는 세력별 조각 상자를 추가합니다.
- 신화 조합 재료는 세력 조합으로도 확장합니다.

예시:

```text
오버워치 신병 + 야전 의무관 + 헬릭스 로켓병 → 솔저 계열 전설 후보
하시모토 말단검객 + 카네자카 감시꾼 + 요괴 자경단원 → 카네자카 전설 후보
정커 고철총잡이 + 고철 폭파수 + 스크랩 정비공 → 정커 챔피언 후보
```
