# 17. 히어로 스트라이크 핵심 재미 개편

## 개편 목적

기존 버전은 스테이지, 강화, 진화, 프로토콜과 영구 성장 시스템은 충분했지만 기본 사격과 명중 자체의 만족감이 약했습니다.

이번 개편은 콘텐츠 수를 더 늘리는 대신 다음 세 가지를 핵심으로 다시 설계합니다.

1. 총을 쏘고 맞히는 순간이 즐거울 것
2. 주무기 세 종류가 실제 운용 방식부터 다를 것
3. 짧은 전투 안에서도 상승과 폭발 구간이 반복될 것

## 핵심 전투 루프

```text
자동사격과 회피
→ 적 처치·그레이즈로 FLOW 축적
→ FLOW 100에서 PULSE RUSH 발동
→ 강화된 화력으로 편대 정리
→ 보스 BREAK 게이지 압박
→ BREAK 취약 시간에 집중 공격
```

## 타격감

### 적 피격 반응

- 피격 순간 흰색 플래시
- 탄환 진행 방향으로 짧은 반동
- 강한 공격일수록 큰 스쿼시·확대 반응
- 레일과 치명타는 피해 숫자 표시
- 적 처치, 엘리트 처치와 보스 BREAK에 서로 다른 화면 흔들림

### 히트스톱

- 일반 명중은 매우 짧은 정지
- 치명타와 레일 명중은 조금 더 강한 정지
- 일반 적 처치, 엘리트 처치와 보스 BREAK는 단계별 정지 시간 사용
- 히트스톱 중에도 UI 피드백은 유지

지속시간은 조작 지연으로 느껴지지 않도록 0.01~0.095초 범위로 제한합니다.

## 주무기 정체성

### 펄스 블래스터

- 공격력 22
- 공격 간격 0.17초
- 빠른 탄속
- 기본 6회 발사마다 좌우 추가탄을 포함한 강화 버스트
- PULSE RUSH 중에는 3회마다 버스트
- 안정적인 연속 처치와 FLOW 유지에 유리

### 산탄 어레이

- 발당 공격력 12
- 공격 간격 0.25초
- 기본 3발 확산
- 짧은 탄환 수명
- 가까운 적에게 최대 약 1.55배 피해
- 멀리 있는 적에게는 피해 감소
- 위험하게 전진할수록 높은 화력을 얻는 무기

### 레일 드라이버

- 공격력 52
- 공격 간격 0.42초
- 매우 빠르고 굵은 탄환
- 기본 관통과 추가 관통
- 강한 반동·히트스톱
- 보스 BREAK 게이지 압박 배율 2.35
- 편대 관통과 보스 제압에 특화

## FLOW와 PULSE RUSH

### FLOW 획득

- 일반 적 처치: 7
- 엘리트 처치: 18
- 보스 처치: 30
- 그레이즈: 3.2
- FLOW 코어: 42
- 보스 BREAK: 24

### FLOW 감소

- 일정 시간 공격적 행동이 없으면 천천히 감소
- 플레이어 피격 시 38 감소
- PULSE RUSH 중 피격하면 남은 지속시간 감소

### PULSE RUSH

FLOW가 100에 도달하면 자동 발동합니다.

- 기본 지속시간 5.6초
- 오버클럭 레벨에 따라 최대 8.85초
- 모든 무기 피해 증가
- 기본탄·드론·미사일 연사 증가
- 이동 반응 속도 증가
- 주무기 추가탄 발사
- 적 처치로 지속시간 소폭 연장
- 전용 금색 오라·속도선·중앙 배너·합성 사운드

기존 오버드라이브 아이템은 즉시 FLOW를 공급하는 FLOW 코어 역할로 변경합니다.

## 보스 BREAK

보스는 체력과 별도로 BREAK 게이지를 가집니다.

- 게이지 최대치는 보스 최대 체력의 18%
- 최소 요구량 420
- 기본탄과 보조 무기가 피해량에 비례해 게이지 압박
- 레일 드라이버는 압박 배율 2.35
- 궁극기와 폭탄도 BREAK 압력 제공

BREAK 발생 시:

- 2.35초 행동·사격 정지
- 화면의 적탄 제거
- 받는 피해 1.38배
- FLOW 24 획득
- 강한 화면 흔들림·히트스톱·전용 사운드
- 종료 후 게이지 최대치의 75%에 해당하는 회복 부하를 먼저 해소해야 다음 BREAK 가능

보스 페이즈 전환 시 BREAK 게이지와 취약 상태를 초기화합니다. 연속 BREAK로 보스를 계속 정지시키는 플레이는 허용하지 않습니다.

## 적 위험 예고

적 공격 직전 외곽 경고 링을 표시합니다.

- 일반 적: 금색
- 저격 적: 분홍색
- 보스: 붉은색

탄환이 나온 뒤에 반응하는 방식뿐 아니라 공격 타이밍을 읽고 미리 이동할 수 있게 합니다.

## 합성 오디오

외부 음원 파일 없이 Web Audio API로 짧은 효과음을 생성합니다.

- 펄스 발사
- 산탄 발사
- 레일 발사
- 일반 명중과 치명타
- 일반 적·엘리트 처치
- 플레이어 피격
- 블링크
- PULSE RUSH
- 보스 BREAK
- 레벨업

모바일 브라우저 정책에 맞춰 첫 화면 터치 후 오디오 컨텍스트를 활성화합니다. 연사음은 종류별 최소 재생 간격을 적용해 소리 중첩과 성능 저하를 방지합니다.

- PULSE RUSH 중 금색 지속시간 게이지
- 현재 수치 또는 남은 시간 표시

### 보스 UI

- 체력 바
- 페이즈 경계선
- BREAK 게이지
- 취약 상태 남은 시간

### 결과 통계

- PULSE RUSH 발동 횟수
- 보스 BREAK 횟수
- 누적 피해
- 최대 콤보
- 피격·블링크
- 목표·무피격 스테이지
- 재추첨 사용량

## 성능 원칙

- 별도 오디오 파일 다운로드 없음
- 오실레이터와 짧은 노이즈 버퍼만 사용
- 피격 반응은 적 객체의 숫자 상태만 갱신
- 히트 파티클은 기존 상한 사용
- BREAK와 FLOW는 별도 물리 객체를 생성하지 않음
- 히트스톱은 프레임 루프를 멈추지 않고 게임 상태 갱신만 잠시 보류

## 검증 항목

- 세 주무기의 발사 형태와 사운드가 서로 다른지
- 산탄의 거리별 피해가 실제로 달라지는지
- 레일이 보스 BREAK를 빠르게 누적하는지
- FLOW 100에서 PULSE RUSH가 한 번만 발동하는지
- 처치로 PULSE RUSH가 연장되는지
- 피격 시 FLOW 또는 지속시간이 감소하는지
- BREAK 중 보스가 이동·사격하지 않는지
- BREAK 종료 직후 연속 기절이 불가능한지
- 페이즈 전환 시 BREAK 상태가 초기화되는지
- 적 공격 직전 경고 링이 표시되는지
- TypeScript 전체 검사와 웹 프로덕션 빌드가 통과하는지
