# 19. 히어로 스트라이크 상용작 기반 전면 재조합

## 목표

히어로 스트라이크를 단순 자동사격 서바이버가 아니라, 세로형 한손 조작에 맞춘 능동 슈팅 로그라이트로 재설계합니다.

핵심 원칙은 다음과 같습니다.

1. 이동과 공격이 같은 효율로 동시에 처리되지 않을 것
2. 강한 화력은 플레이어의 정렬과 타이밍 결정에서 나올 것
3. 전투 중 성장 화면이 자주 흐름을 끊지 않을 것
4. 웨이브마다 다른 전술 요구가 있을 것
5. HUD는 생존과 전투 판단에 필요한 정보만 보여줄 것
6. 장기 성장은 숫자 인플레이션보다 선택지 해금에 가깝게 유지할 것

## 상용작에서 가져오는 구조

### Archero

- 이동과 공격을 분리해 피하기와 반격의 리듬을 만듭니다.
- 방과 장애물, 적 조합이 공격 타이밍을 결정하게 합니다.

### 20 Minutes Till Dawn

- 조준과 발사를 플레이어 행동으로 남깁니다.
- 일반 성장과 보스의 대형 보상을 분리합니다.

### Brotato

- 짧은 웨이브와 웨이브 사이 빌드 결정을 분리합니다.
- 자동사격을 사용하더라도 전투와 선택이 명확한 박자를 가집니다.

### Sky Force Reloaded

- 스테이지 진행도와 임무 목표가 반복 플레이 동기를 만듭니다.
- 전투 중 HUD는 임무와 생존 정보를 우선합니다.

### Nova Drift

- 무기, 실드, 기체가 단순 스탯이 아니라 플레이 방식을 바꿉니다.

### Deep Rock Galactic: Survivor

- 자동사격 외의 이동 목적을 제공합니다.
- 전투 중 임무 목표가 위치 선택을 바꿉니다.

## 새 조작 모델

### DRIVE

손가락을 누르고 끄는 동안의 이동 상태입니다.

- 빠른 이동 반응
- 주무기 제압 사격
- 제압 사격 피해 55%
- 제압 사격 간격 1.35배
- 보조무기 효율 75%
- 레일 드라이버 충전
- 피격 회피와 목표 정렬에 사용

### FOCUS

손가락을 놓은 뒤 0.14초가 지나면 시작됩니다.

- 플레이어 이동 감속
- 현재 X축과 가장 가까운 적 락온
- 락온 허용 폭 기본 76px
- 주무기별 정밀 사격 활성화
- 약점과 엘리트 우선 락온
- 다시 드래그하면 즉시 DRIVE로 전환

### 무기별 정밀 사격

#### 펄스 블래스터

- DRIVE: 빠른 단발 제압 사격
- FOCUS: 3연속 락온 버스트
- 같은 목표를 계속 맞히면 마지막 탄 피해 증가

#### 산탄 어레이

- DRIVE: 좁은 2발 견제 사격
- FOCUS: 근거리 집중 산탄
- 락온 대상이 가까울수록 펠릿 집중도와 피해 증가

#### 레일 드라이버

- DRIVE: 레일 충전, 약한 보조탄
- FOCUS 진입: 충전량을 소비해 관통 레일 발사
- 완전 충전 시 약점 피해와 BREAK 압력 증가

## 전투 구간

각 스테이지는 네 개의 명명된 구간과 보스로 구성합니다.

1. SCOUT — 기본 적과 조준 학습
2. CROSS FIRE — 좌우 협공과 정렬 변경
3. HUNTER — 돌진 적과 저격 적 중심
4. SIEGE — 탱커, 폭격기, 엘리트 압박
5. BOSS — 약점 정렬과 BREAK

구간 전환 시:

- 0.7초 진입 배너
- 적탄 일부 정리
- 대기 중인 강화가 있으면 강화 화면 표시
- 편대 패턴을 새 구간 규칙으로 교체

## 성장 리듬

### 경험치

- 경험치는 전투 중 계속 획득합니다.
- 요구량을 채워도 즉시 전투를 멈추지 않습니다.
- 대기 강화 수를 최대 2개까지 저장합니다.
- 강화 선택은 웨이브 경계 또는 보스 진입 전에만 표시합니다.
- 한 경계에서 한 번만 선택합니다.

### 일반 강화

일반 강화는 8~12회 선택을 목표로 합니다.

- 주무기 성능
- 조준 폭과 락온 속도
- 보조무기
- 생존
- 블링크와 궁극기

### 핵심 모듈

보스 처치 후에는 일반 스탯이 아니라 플레이 방식을 바꾸는 핵심 모듈을 선택합니다.

- FOCUS ASSAULT — 정밀 사격 강화, 이동 사격 약화
- MOBILE ARSENAL — 이동 사격 강화, 락온 폭 축소
- BREAK HUNTER — 약점과 BREAK 특화
- AEGIS RUNNER — 블링크와 방어 중심
- SWARM COMMAND — 보조무기 중심이지만 락온 목표를 공유

## 적 설계

- 러너: DRIVE 상태의 이동 경로를 읽고 돌진
- 드론: 락온을 분산시키는 소형 편대
- 탱커: 정면 방어, 측면 정렬 요구
- 저격기: FOCUS 상태를 오래 유지하면 조준 사격
- 위버: X축 정렬을 흔드는 교차 이동
- 폭격기: 머물 공간을 제한하는 지연 폭발
- 엘리트: 가장 높은 락온 우선순위와 고유 특성
- 보스: 이동 약점, BREAK 취약 구간, 페이즈별 조준 요구

## HUD 우선순위

### 항상 표시

- 스테이지와 현재 전투 구간
- 네 칸 구간 진행도
- HP와 보호막
- DRIVE / FOCUS 상태
- 락온 대상 표시
- 블링크와 궁극기
- 보스 HP와 BREAK

### 전투 중 숨김

- 점수
- 연구 등급
- 프로토콜 개수
- 누적 피해
- 상세 성장 수치

이 정보는 결과 화면과 일시정지 화면에서 확인합니다.

## 삭제 또는 축소

- TEMPO 피해 배율 삭제
- 정지 포격 삭제
- 전투 중 점수 HUD 삭제
- 매 스테이지 프로토콜 선택 삭제
- 모든 무기가 같은 자동사격 규칙을 공유하는 구조 삭제
- 경험치 도달 즉시 강화 화면을 여는 구조 삭제

## 목표 수치

- 전체 런: 10~13분
- 스테이지 일반전: 48~62초
- 일반 강화: 8~12회
- 핵심 모듈: 3~4회
- DRIVE 이동 사격 피해: FOCUS 기준 55~70%
- 보조무기 기여도: 전체 피해 20~30%
- FOCUS 진입 시간: 0.14초
- 락온 폭: 76px, 강화로 최대 116px
- 웨이브 경계 전환: 0.7초

## 검증 기준

- 손을 계속 끌 때와 놓을 때 무기 리듬이 즉시 다르게 느껴지는지
- 가만히 있는 것만으로 모든 적이 자동 정리되지 않는지
- 움직임이 단순 페널티 회피가 아니라 조준과 공격 준비에 연결되는지
- 강화 화면이 전투 중간을 자주 끊지 않는지
- 네 구간이 적 조합과 진입 배너로 구분되는지
- HUD에서 적탄과 플레이어 상태가 가려지지 않는지
- TypeScript 검사와 프로덕션 웹 빌드가 통과하는지
