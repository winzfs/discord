# 28. 전투 플로팅 텍스트 표시 규칙

## 1. 목적

몬스터 타격 시 뜨는 텍스트를 더 읽기 쉽게 정리했습니다.

요청 기준:

```text
스킬 사용 텍스트는 신화급만 표시
일반 데미지는 빨간색
치명타 데미지는 노란색
보상은 녹색
```

## 2. 적용 파일

```text
apps/web/src/game-client/pixi/pixiFloatingTextView.ts
```

## 3. 중앙 처리 방식

전투 텍스트는 여러 런타임에서 `floatText()`를 통해 출력됩니다.

```text
pixiCombatRuntime.ts
pixiSkillRuntime.ts
pixiBaseHeroSkillRuntime.ts
pixiHeroAttackFxRuntime.ts
pixiControlActionRuntime.ts
```

각 파일을 따로 고치면 누락이 생길 수 있으므로, 실제 텍스트 객체를 만드는 `pixiFloatingTextView.ts`에서 중앙 규칙으로 정리했습니다.

## 4. 색상 규칙

### 4.1 데미지

숫자만 있는 텍스트는 데미지로 봅니다.

```text
예: 138, 426, 1200
```

기본 색상:

```text
빨간색 0xff4b4b
```

### 4.2 치명타

기존 호출 색상이 치명타 계열 노란색이면 치명타로 봅니다.

```text
0xffd166
0xfff06a
0xffe066
0xffff66
```

치명타 색상:

```text
노란색 0xffd166
```

### 4.3 보상

다음 텍스트는 보상으로 봅니다.

```text
+숫자
보상 포함 텍스트
행운석 + 포함 텍스트
```

보상 색상:

```text
녹색 0x5ee27a
```

## 5. 스킬 텍스트 규칙

기본 유닛 스킬에서 뜨던 짧은 라벨은 숨깁니다.

```text
공격
약점
폭발
연쇄
관통
영역제어
단일빙결
단일제어
표식
지원
포탑
보상
반격
```

신화 스킬 텍스트는 `pixiSkillRuntime.ts`에서 신화 유닛일 때만 발생하므로 그대로 표시합니다.

## 6. 테스트 체크리스트

- 일반 데미지 숫자가 빨간색으로 뜨는지 확인
- 캐서디/강한 치명 계열 데미지가 노란색으로 뜨는지 확인
- 골드 보상 `+숫자`가 녹색으로 뜨는지 확인
- 행운석 보상이 녹색으로 뜨는지 확인
- 일반/희귀/영웅/전설 유닛의 짧은 스킬 라벨이 숨겨지는지 확인
- 신화 유닛의 스킬명 텍스트는 계속 뜨는지 확인
