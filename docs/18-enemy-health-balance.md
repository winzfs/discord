# 18. 몬스터 체력 밸런스 노트

## 1. 목적

영웅 공격력, 레벨 성장, 연계 시너지, 스킬 효과가 누적되면서 몬스터가 너무 빨리 녹는 문제가 있었습니다.

이번 조정은 웨이브 배율이 아니라 `packages/game/src/data/enemies.ts`의 기본 체력 자체를 다시 상향하는 작업입니다.

## 2. 현재 기본 체력

| 몬스터 | 이전 기본 체력 | 현재 기본 체력 |
|---|---:|---:|
| 잡버그 | 180 | 320 |
| 핑러너 | 150 | 260 |
| 렉덩어리 | 620 | 1100 |
| 엘리트버그 | 1150 | 2100 |
| 서버크래셔 | 4200 | 7600 |

## 3. 유지한 항목

이번에는 다음 값은 변경하지 않았습니다.

- 몬스터 속도
- 처치 보상
- 누수 피해
- 웨이브 스폰 수
- 웨이브 체력 배율
- 이자 시스템

## 4. 현재 체력 배율

기본 체력 위에 아래 배율이 적용됩니다.

```ts
hpScale = 1.75 + wave * 0.24 + (boss ? 1.35 + wave * 0.34 : 0)
```

따라서 실제 체력은 다음처럼 계산됩니다.

```text
실제 체력 = 기본 체력 * 웨이브 체력 배율
```

## 5. 테스트 기준

확인할 항목:

- 1~3웨이브에서 너무 빠르게 녹지 않는지
- 5웨이브 전후에 첫 누수 압박이 생기는지
- 보스가 최소 몇 초 이상 버티는지
- 초반 소환/강화 선택이 의미 있게 갈리는지
- 지원 버프 배치가 강하지만 필수 정답이 되지는 않는지

## 6. 다음 조정 후보

여전히 쉽다면 다음 순서로 조정합니다.

1. 기본 체력 추가 상향
2. 보스 기본 체력 별도 상향
3. 웨이브 스폰 수 증가
4. 빠른 몬스터 속도 소폭 증가
5. 보상량은 그대로 두고 강화 비용 증가

반대로 너무 어렵다면 다음 순서로 되돌립니다.

1. 1~3웨이브 기본몹 체력만 하향
2. 핑러너 체력만 하향
3. 보스 추가 배율만 하향
